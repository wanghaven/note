

## 1. 概述

Obsidian Copilot 是一款将大型语言模型（LLM）无缝集成到 Obsidian 中的开源插件，旨在成为用户的“第二大脑”中的终极 AI 助手。它注重本地化和隐私保护，允许用户在自己的知识库中与 AI 进行交互，从而激发灵感、连接知识点并高效创作。

该插件分为免费版和付费版（Plus/Believer），不同版本在功能和体验上有显著差异。本笔记将深入探讨各个版本的功能，并通过详细的表格进行对比。

## 3. 核心功能深入解读

#### **基础功能 (免费版)**

*   **自带 API Key (Bring your own API key)**: 这是免费版的核心。用户需要拥有并配置自己的 OpenAI、Anthropic 或其他兼容服务的 API Key。这意味着 AI 调用的成本由用户自己承担。同时，这也提供了灵活性，用户可以选择使用本地部署的模型（通过 LM Studio 或 Ollama）来实现离线使用。
*   **一流的 LLM 聊天体验**: 提供了一个无干扰的侧边栏聊天界面，可以与 AI 进行流畅的对话。支持保存聊天记录、一键复制答案等便捷操作。
*   **本地数据存储 (Local data store for Vault QA)**: 当你使用 Vault QA（知识库问答）功能时，插件会在你的本地设备上为你的笔记创建索引（向量数据库）。这确保了你的笔记内容不会被上传到任何云服务商，保护了数据隐私。
*   **视觉模型图像支持**: 如果你配置的模型支持多模态（如 GPT-4o），你可以直接在聊天中引用笔记中的图片，让 AI 理解并回答与图片相关的问题。

#### **Plus 版高级功能**

*   **专属 AI 模型**: Plus 用户无需配置自己的 API Key 即可使用内置的 `copilot-plus-flash` 聊天模型和两种嵌入模型（`small` 和 `multilingual`）。这不仅简化了设置，也可能因为模型的优化而带来更好的性能和多语言支持。
*   **AI 代理和工具使用 (AI Agent and Tool Use)**: 这是付费版与免费版最核心的区别之一。它将 Copilot 从一个简单的聊天机器人升级为一个智能代理。AI 可以根据你的问题，自主判断是否需要使用工具来辅助回答。
    *   **网络搜索 (`@web`)**: 当你询问一个需要最新信息的问题时（例如“今天的天气怎么样？”），AI 代理会自动调用网络搜索工具，整合搜索结果后给你答案。
    *   **YouTube 转录 (`@youtube`)**: 你只需提供一个 YouTube 链接，AI 代理就能快速获取视频的完整文字稿，并可以基于文字稿进行总结、提炼要点或回答特定问题。
*   **非 Markdown 文件支持**: 极大地扩展了 Obsidian 作为知识库的边界。你可以将 PDF 格式的学术论文、EPUB 格式的电子书直接放入 Vault，并让 Copilot 对这些长文档进行深度问答，这对于研究和学习非常有价值。
*   **项目模式 (Projects mode)**: 允许你定义不同的“上下文”。例如，你可以创建一个“工作”项目，只包含与工作相关的文件夹和标签。当切换到这个项目时，AI 的所有交互都将局限在这个范围内，确保了回答的高度相关性，避免了知识库内容的交叉干扰。

#### **Believer 版独有价值**

*   **终身访问权限**: 对于重度用户和 Obsidian 的长期主义者来说，这是一个极具吸引力的选项。一次性付费意味着锁定了未来的所有功能更新，避免了持续订阅的费用。
*   **最强嵌入模型 (`copilot-plus-large`)**: Believer 用户独享一个更大、更强大的嵌入模型。理论上，这意味着在进行全库知识问答时，AI 对你笔记语义的理解会更深刻，从而提供更精准、更相关的答案。
*   **离线模式 (Offline mode)**: 这是最受期待的功能之一，尤其对于注重隐私的用户。未来，Believer 用户将能够配置插件使用完全在自己电脑上运行的本地大语言模型。这将实现所有 AI 功能的完全离线操作，数据永远不会离开本地设备，提供了最高级别的隐私和安全保障。
*   **优先权**: 作为项目的最高级别支持者，Believer 用户在功能请求和技术支持上拥有最高优先级，能够更早地体验到插件的最新成果，并对项目的发展方向产生影响。
