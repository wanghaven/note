## 2. 摘要 (Abstract)

本文针对配备大规模 MIMO 天线技术的卫星，研究了线性预编码 (LP) 与基于码本的波束赋形的联合设计方案。该方案的波束赋形权重码本由离散傅里叶变换 (DFT) 矩阵的列向量构成，旨在受限的发射功率条件下最大化系统可实现的吞吐量。相应的优化问题最初被表述为一个混合整数非线性规划 (MINP) 问题。为了有效解决这一极具挑战性的非凸优化问题，本文利用加权最小均方误差 (WMMSE) 变换、拉格朗日对偶法以及匈牙利算法 (Hungarian algorithm) 等数学工具，开发了一种高效的联合迭代优化算法。此外，本文还提出了一种贪婪算法作为性能对标基准。报告提供了上述方案的复杂度分析，并通过广泛的蒙特卡洛仿真验证了所提算法在提升系统容量方面的优越性。

--------------------------------------------------------------------------------

## 3. 引言 (I. Introduction)

### 3.1 背景与需求

- **中低轨卫星优势：** 低轨 (LEO) 和中轨 (MEO) 卫星在提供普遍网络接入及支持物联网数据回传方面具有显著潜力，是未来非地面网络 (NTN) 的重要组成部分。
- **现有技术局限：** 尽管全频率复用预编码在地面系统中已臻成熟，但直接将其应用于卫星 Massive MIMO 面临巨大的挑战。现有研究虽然提出了基于统计信道状态信息 (CSI) 的预编码，但其对星载硬件的计算资源消耗过高。
- **DFT 波束赋形的动机：** DFT 技术通过快速傅里叶变换 (FFT) 实现，能够以较低的硬件复杂度生成多个独立波束，显著降低 OBP 的质量和功耗。然而，传统 DFT 波束赋形缺乏动态指向的灵活性。

### 3.2 本文主要贡献

- **联合架构设计：** 首次提出在 DFT 波束赋形框架之上叠加线性预编码 (LP) 的混合处理架构，实现了空间增益与干扰抑制的平衡。
- **数学建模与转换：** 构建了兼顾复值变量（LP）和二元变量（DFT 向量选择）的 MINP 优化模型，并证明了其向 WMMSE 问题的等价转化。
- **高效算法开发：**
    - **迭代优化：** 利用 WMMSE 框架将非凸速率最大化问题转化为对各变量组分别凸的形式。
    - **闭式解推导：** 应用拉格朗日对偶法导出了 LP 权重的闭式解。
    - **指派问题转化：** 将离散的 DFT 向量匹配问题转化为经典的"指派问题"，并利用匈牙利算法获得最优解。
- **低复杂度基准：** 提出了基于最大增益搜索的贪婪算法。

## 4. 系统模型与问题表述 (II. System Model and Problem Formulation)

### 4.1 OBP 架构描述

根据 Figure 1，星载处理器 (OBP) 架构由以下六个关键部分组成：

1. **MODCOD 模块：** 执行前向纠错 (FEC) 编码与符号调制。
2. **线性预编码 (LP)：** 应用预编码矩阵 $\mathbf{U} \in \mathbb{C}^{M \times M}$ 处理 $M$ 个用户流。
3. **DFT 波束匹配：** 本项目的核心创新点，确定每个信号流与 DFT 码本向量的映射关系。
4. **DFT 波束赋形：** 基于 $N \times N$ 的 DFT 矩阵 $\mathbf{W}$。
5. **空间窗口与辐射单元选择：** 选取 $K \le N$ 个 DFT 输出，以适应天线阵列规模并调整波束宽度。
6. **模拟接口：** 包含频率上变频、功率放大（PA）及 RF 链。

**LP 与 DFT 波束匹配功能对比表：**

|   |   |   |
|---|---|---|
|功能项|线性预编码 (LP)|DFT 波束匹配|
|**数学性质**|连续复数值变量 $\mathbf{U}$|二元整数变量 $\mathbf{A}$|
|**物理意义**|调整信号功率分配与相位，消除 MUI|将信号流映射至特定的 FFT 输入端口|
|**设计目标**|多用户干扰消除与动态波束转向|选取最佳空间方向向量以增强有用信号|

### 4.2 数学建模

定义二元指派矩阵 $\mathbf{A} \in \mathbb{R}^{N \times M}$，其元素 $a_{n,m}$ 表示第 $n$ 个 DFT 向量是否分配给第 $m$ 个流：

$$a_{n,m} = \begin{cases} 1, & \text{若第 } n \text{ 个 DFT 向量分配给符号 } s_m \\ 0, & \text{否则} \end{cases} \tag{1}$$

物理上，矩阵 $\mathbf{A}$ 的约束体现了星载硬件中信号流与 FFT 端口的一一映射限制：

$$(C1) : \sum_{\forall n} a_{n,m} = 1, \quad \forall m \tag{2}$$

$$(C2) : \sum_{\forall m} a_{n,m} \le 1, \quad \forall n \tag{3}$$

DFT 块的输入和输出分别表示为：

$$\tilde{\mathbf{s}} = \mathbf{As} = \mathbf{AUx} \tag{4}$$

$$\mathbf{W}\tilde{\mathbf{s}} = \mathbf{WAs} = \mathbf{WAUx} \tag{5}$$

### 4.3 SINR 建模与问题表述

- **SINR 表达式：** 用户 $m$ 的信号与干扰加噪声比（SINR） $\Gamma_m$ 定义为（其中 $\sigma_m^2$ 为接收端噪声方差）：

$$\Gamma_m(\mathbf{U}, \mathbf{A}) = \frac{\|\tilde{\mathbf{h}}_m^H \mathbf{W A} \mathbf{u}_m\|^2}{\sum_{j \neq m} \|\tilde{\mathbf{h}}_m^H \mathbf{W A} \mathbf{u}_j\|^2 + \sigma_m^2} \tag{7}$$

- **目标函数：** 在整星射频功率预算 $P$ 约束下最大化系统和速率：

$$\max_{\mathbf{U}, \mathbf{A}} \sum_{\forall m} \log_2(1 + \Gamma_m(\mathbf{U}, \mathbf{A})) \tag{8a}$$

$$\text{s.t. } (C1),\ (C2),\  (C3): \text{Trace}(\mathbf{U}^H\mathbf{U}) \le P \tag{8b}$$

该问题属于非凸 MINP，且 $\mathbf{U}$ 与 $\mathbf{A}$ 之间存在高度非线性耦合。

## 5. 核心难点深度解析一：WMMSE 转换机制

根据定理 1，非凸的和速率优化可通过引入辅助变量转换为 WMMSE 形式。

- **等效目标函数：**

$$\min_{\mathbf{A}, \mathbf{U}, \{\delta_m, \omega_m\}} \sum_{\forall m} (\omega_m e_m - \log \omega_m - 1) \tag{9}$$

- **关键参数解析：**
    1. **接收系数** $\delta_m$：其最优解即为 MMSE 解 $\delta_m^* = \delta_{\text{MMSE}}$（公式(10)），反映了接收端的滤波处理。
    2. **MSE 权重** $\omega_m$：建立了速率与均方误差之间的倒数关系，最优时 $\omega_m^* = e_m^{-1} = 1 + \Gamma_m$（公式(11)）。
- **转换意义：** 这一转换将分母中包含干扰项的复杂对数函数拆解为对各变量组准凸的形式，为交替优化策略铺平了道路。

--------------------------------------------------------------------------------

## 6. 核心难点深度解析二：基于拉格朗日对偶法的 LP 设计

在固定分配矩阵 $\mathbf{A}$ 和权重的情况下，LP 的设计变为一个凸二次规划问题。

- **拉格朗日函数构建：**

$$\mathcal{L}(\mathbf{U}, \beta) = \sum_{m} \left[ \mathbf{u}_m^H (\mathbf{\Theta} + \beta \mathbf{I}) \mathbf{u}_m - 2\omega_m \text{Re}(\mathbf{k}_m^H \mathbf{u}_m) \right] - \beta P \tag{13}$$

- **闭式解推导：** 通过 KKT 条件，令 $\nabla_{\mathbf{u}_m} \mathcal{L} = 0$，得到最优预编码列向量：

$$\mathbf{u}_m^* = (\mathbf{\Theta} + \beta \mathbf{I})^{-1} \mathbf{k}_m \omega_m \tag{14}$$

- **对偶变量更新：** 使用次梯度法迭代搜索最优拉格朗日乘子 $\beta$：

$$\beta^{[\ell+1]} = \left[ \beta^{[\ell]} + r^{[\ell]} \left(\text{Trace}(\mathbf{U}^H \mathbf{U}) - P\right) \right]^+ \tag{15}$$

该过程确保了在充分利用发射功率的同时，精准控制多用户干扰。

--------------------------------------------------------------------------------

## 7. 核心难点深度解析三：匈牙利算法与指派矩阵构建

**定理 2 的深度洞察：** 对于离散变量 $\mathbf{A}$，原始问题呈现出复杂的二次形式 $\mathbf{a}_n^T \mathbf{\Psi}_n \mathbf{a}_n$。

- **等价性简化：** 由于 $a_{n,m}$ 的二元属性及其范数约束 $\|\mathbf{a}_n\|_1 \le 1$，根据定理 2，二次项可以简化为线性项 $\text{diag}(\mathbf{\Psi}_n)^T \mathbf{a}_n$。
- **深度解析：** 这种从二次二元规划到线性指派问题的转化，本质上源于 **DFT 矩阵列向量的正交性**。正交性消除了指派变量间的交叉耦合，使得全局最优解可以通过匈牙利算法在多项式时间内求得。
- **权重矩阵：** 构造指派权重 $\rho_{n,m} = [\text{diag}(\mathbf{\Psi}_n) - 2\mathbf{f}_n]_m$，其中包含了信道增益与潜在干扰的加权信息。

## 8. 算法实现与复杂度分析

### 8.1 算法流程对比

- **Algorithm 1 (迭代法)：** 采用全参数交替优化（Weights -> LP -> Hungarian A），收敛性由目标函数的单调递减特性保证。
- **Algorithm 2 (贪婪法)：** 基于单一最大空间增益匹配 $\mathbf{A}$，随后采用零迫 (ZF) 预编码。

### 8.2 复杂度分析

|   |   |   |
|---|---|---|
|算法|复杂度量级|关键驱动因素|
|**Algorithm 1**|$\mathcal{O}(b_{\text{Alg.1}}^{-1} \times N^3)$|由迭代精度 $b_{\text{Alg.1}}$ 及匈牙利算法决定|

## 10. 结论与总结

### 10.1 结论

本文提出了一种适用于 OBP 卫星的联合 LP 与 DFT 波束赋形架构。通过 WMMSE 转换和指派问题建模，所提算法在保持较低星载计算开销的同时，实现了接近最优的抗干扰性能与容量提升。

### 10.2 学术总结与评价

**综合评价：** 本文在星载信号处理领域展示了"算法复杂度与系统效能"平衡的极高艺术。其核心价值在于以下三点：

1. **工程可行性：** 通过 DFT/FFT 架构避开了高功耗的复数矩阵直接乘法，完美契合 EGERTON 项目对于低功耗星载处理的要求。
2. **理论创新：** 巧妙地利用 DFT 码本的正交特性，将原本 NP-hard 的 MINP 问题简化为多项式时间可解的指派问题，这一处理逻辑对未来 6G 非地面网络 (NTN) 的混合波束赋形设计具有极强的借鉴意义。
3. **性能突破：** 仿真结果有力地反驳了"混合架构性能必弱于全数字"的直觉，证明了在特定约束条件下，联合优化所带来的自由度增益足以抵消码本量化带来的损失。
