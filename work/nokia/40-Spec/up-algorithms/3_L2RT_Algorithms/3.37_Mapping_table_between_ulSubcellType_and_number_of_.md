# 3.37 Mapping table between ulSubcellType and number of antennas

## 3.37.0-1 Mapping table between ulSubcellType and number of antennas 7-2a (ID: 11217560)

**3.37.0-1.0-1**  (ID: `11217568`)

Mapping table between ulSubcellType and number of antennas
The supported subcell types and corresponding number of antennas on different channels are specified as below:

If NIDD:actEcpri72e = false:

|  |  |  |  |  |
| --- | --- | --- | --- | --- |
| ulSubcellType in DU | Number of spatial streams |  |  |  |
| PUSCH | SRS | PUCCH | PRACH |  |
| C4 | 4 | 4 | 4 | 4 |
| C2 | 2 | 2 | 2 | 2 |
| C8 | 8 | 8 | x | 8 |

NOTE: x = 4 if NIDD:numOfPUCCHRx is equal to 4; x= 8 if NIDD:numOfPUCCHRx is equal to 8.
