# 5.4 ML based remote interference detection

## 5.4.0-1 ML based remote interference detection ML based RI detection Algorithm Overview (ID: 13472991)

**5.4.0-1.0-1**  (ID: `13472992`)

ML based remote interference detection
ML based RI detection uses a DNN model to detect remote interference based on UL RSSI counter data in state 1 ([11161340](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktT6svvSEe-AqvopbP1qhQ?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9)) and state 2 ([11161371](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktT6t_vSEe-AqvopbP1qhQ?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9)).

In each RI monitoring window rdInputDataPeriodForRI, L2-PS collects UL RSSI per second for UL RSSI measurements.

The smoothing shall be performed within subband per UL symbol in each slot ([13482853](https://dn-prod.ext.net.nokia.com/rm/resources/BI_5LAPkD8WEfG0jPRAdLhzvA?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9)). Smoothed records shall be reset when RI monitoring period (nrRiMonitorWindow) restarts.

L2-PS sends collected UL RSSI measurements via inference request to ML inference engine to perform RI prediction.

At the end of each RI monitoring window, gNB will calculate RISlotRatio:

RISlotRatio = RI\_true\_num/ Total\_RI\_num.

Total\_RI\_num = nrRiMonitorWindow/rdInputDataPeriodForRI

gNB compares RISlotRatio to threshold rdNrRiThresRatioForAI. If RISlotRatio exceeds threshold, AIML based RI is confirmed for this session.

## 5.4.0-4 ML based remote interference detection Smoothing interference measurement result from L1 for ML based RI detection (ID: 13482853)

**5.4.0-4.0-1**  (ID: `13482854`)

ML based remote interference detection
5G-L2-PS shall smooth interference measurement result reported by 5G-L1-UL.

* The recorded values shall be smoothed using the following formula:

RI[i][symbol]=RI’[i][symbol]*(1-rdRimSmoothAlpha(5G\_UP\_ALG\_9655\_replaced([11163724](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktbPdfvSEe-AqvopbP1qhQ))) )+RI\_L1H*rdRimSmoothAlpha(5G\_UP\_ALG\_9655\_replaced([11163724](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktbPdfvSEe-AqvopbP1qhQ))),

where i indicates slot index (see 5G\_UP\_ALG\_9328\_replaced([11161420](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktUhwfvSEe-AqvopbP1qhQ))), RI’ is previously measured RI in the same slot and symbol, *rdRimSmoothAlpha (5G\_UP\_ALG\_9655\_replaced([11163724](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktbPdfvSEe-AqvopbP1qhQ))) shall be configurable R&D parameter, with default value of 0.1.*

* If rdRiMonitorAllSsb="1" (see 5G\_UP\_ALG\_31909\_replaced([11163711](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktbPc_vSEe-AqvopbP1qhQ))), gNB-DU shall perform the smoothing per subband per configured SSB beam and per UL symbol in each slot,   
  else if rdRiMonitorAllSsb="0" ,  The **smoothing shall be performed within subband per UL symbol in each slot.**
* Smoothed record shall be reset when RI monitoring period (*NRRIMonitorWindow) restarts, initial value shall be**defined by RI detection threshold ([12713680](https://dn-prod.ext.net.nokia.com/rm/resources/BI_PGfkwWhmEfCk8v_SG2h2xg))*.\*\*\*\*

[PR863062]

* (smooth for RI pilot window)When NRCELL/NRRIM\_PROFILE/actRiPilotDlMute = true, if the RI pilot window (given by NRCELL/NRRIM\_PROFILE/riPilotWindow) <=10, and when the the RI pilot window starts, gNB-DU shall create the separate smoothed record sets with initial value from the first measurement reports the RI pilot window.
* gNB-DU shall maintain the separate smoothed record sets using the following formula:
* RI[i][symbol]=RI’[i][symbol]*0.5+RI\_L1H*0.5
* where，i indicates slot index, RI’ is the previous measured RI in the same slot and the same symbol.

[End PR863062]

Smoothed record shall be reset when RI monitoring period (NRRIMonitorWindow) restarts, initial value shall be defined by RI detection threshold(12713680).

Due to the measurement result smoothing, 5G-L2-PS shall deduce the beam ID value for the RI[i][symbol] based on below criteria:

- In case PRB range and the UL symbol index of RI\_L1H are NOT used for any UL channel during the monitoring window

Beam ID value shall be the pattern ID of RI beam **(see 5G\_UP\_ALG\_31912\_replaced([11161588](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ktUhzPvSEe-AqvopbP1qhQ)))**

- In case PRB range and the UL symbol index of RI\_L1H are from UL scheduling with same sector beam ID during the monitoring window

Beam ID value shall be the sector beam ID  (see 5G\_UP\_ALG\_2037\_replaced([11112372](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpVZp_vSEe-AqvopbP1qhQ)))

- In case PRB range and the UL symbol index of RI\_L1H are used for scheduling with different beam pattern IDs during the monitoring window

Beam ID value shall **has no value  in case no beam was used or no measurement was requested.**

## 5.4.0-7 ML based remote interference detection DNN (Deep Neural Network) (ID: 13473078)

**5.4.0-7.0-1**  (ID: `13473079`)

ML based remote interference detection
Input layer: 160 symbols‘ RSSI for each sample (1 request contains up to 160 symbol level RSSI UL counter values)

2 hidden layers (256+128 neurons)

Output Layer: 1 neuron with sigmoid activation which indicates 0 or 1 (binary classification)
