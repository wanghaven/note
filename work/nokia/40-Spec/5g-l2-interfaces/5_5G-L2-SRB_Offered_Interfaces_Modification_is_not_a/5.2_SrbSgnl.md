### 5.2.2.4 Short MAC-I Calculation

#### 5.2.2.4.0-1 Short MAC-I Calculation General (ID: 10993032)

**5.2.2.4.0-1.0-1**  (ID: `10993038`)

Short MAC-I Calculation
Messages:

SrbSgnl\_ShortMacCalculationReq/Resp

Purpose:

To request shortMAC-I calculation

Precondition:

Security has been activated in L2-SRB

Addressing:

The request is sent to the SrbSgnl address.

The response is sent to the address where the request was sent from.

Relevant UP/CP MSCs:

5G\_L1L2\_FLOW\_3594\_replaced([11184104](https://dn-prod.ext.net.nokia.com/rm/resources/BI_Be0pL_vREe-AqvopbP1qhQ))

---

#### 5.2.2.4.0-2 Short MAC-I Calculation Description (ID: 10993044)

**5.2.2.4.0-2.0-1**  (ID: `10993050`)

Short MAC-I Calculation
The request contains RRC message data and cell identities for short integrity calculations.

Resp includes calculated short integrity results.
