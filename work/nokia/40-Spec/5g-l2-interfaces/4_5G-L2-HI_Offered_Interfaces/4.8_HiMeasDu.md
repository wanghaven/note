### 4.8.2.1 L2-HI DU Load Measurement Setup

#### 4.8.2.1.0-1 L2-HI DU Load Measurement Setup Overview (ID: 10991231)

**4.8.2.1.0-1.0-1**  (ID: `13459837`)

L2-HI DU Load Measurement Setup
Messages:

HiMeasDu\_LoadMeasurementSetupReq/Resp

Purpose:

To start the load measurements to be used for 5G-L2-HI (DU) instance selection.

Precondition:

5G-L2-HI (DU) instance has been started and HiCnfgDu\_AddressDistributionReq is received.

Addressing:

The request is sent to the HiMeasDu address attached to the corresponding 5G-L2-HI (DU) instance.

The response is sent to the address where the request was sent from.

Relevant UP MSCs:

N/A

---

#### 4.8.2.1.0-2 L2-HI DU Load Measurement Setup Description (ID: 10991239)

**4.8.2.1.0-2.0-1**  (ID: `13459839`)

L2-HI DU Load Measurement Setup
Upon reception of the request, 5G-L2-HI shall store the measurement configuration and start the load measurement as per 5G\_UP\_ALG\_13834.

---

#### 4.8.2.1.0-3 L2-HI DU Load Measurement Setup Message validation (ID: 10991244)

**4.8.2.1.0-3.0-1**  (ID: `13459840`)

L2-HI DU Load Measurement Setup
Upon reception of the HiMeasDu\_LoadMeasurementSetupReq message, 5G-L2-HI shall check the validity of the request message.

5G-L2-HI shall reply with HiMeasDu\_LoadMeasurementSetupResp message and set the status IE = Ok if the request message is valid and measurement was configured successfully.

For exception handling, see 5G\_L2\_IF\_1257\_replaced([10991251](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2Wwxkvu6Ee-AqvopbP1qhQ)).

---

#### 4.8.2.1.0-4 L2-HI DU Load Measurement Setup Exceptions (ID: 10991251)

**4.8.2.1.0-4.0-1**  (ID: `13459841`)

L2-HI DU Load Measurement Setup
Expected Exceptions:

[5GC002184-G]

Generic exception handling rules can be found in the chapter 5G\_UP\_1795\_replaced([11374912](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueKrmfvkEe-AqvopbP1qhQ)).

5G-L2-HI shall reply with HiMeasDu\_LoadMeasurementSetupResp message, set the status IE = NotOk, and set an appropriate cause value if the request message is not valid or if the measurement was not configured successfully.

For available cause values, see 5G\_UP\_1846\_replaced([11375790](https://dn-prod.ext.net.nokia.com/rm/resources/BI_uePkGPvkEe-AqvopbP1qhQ)).

[End 5GC002184-G]

Unexpected Exceptions:

[Before 5GC002184-G]

In case of unexpected exception, no report is sent.

[End 5GC002184-G]

[5GC002184-G]

In case of an unexpected exception, 5G-L2-HI shall not send response message.

[End 5GC002184-G]

[5GC002184-G]

Error is logged for both expected and unexpected exception and report (5G\_L2\_IF\_1261\_replaced([10991295](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2Wwxmfu6Ee-AqvopbP1qhQ)) HiMeasDu\_LoadMeasurementRepInd) is not sent.

[End 5GC002184-G]
