#### 2.11.1.4.0-2 PsTrace Pcmd Timing Advance Indication Description (ID: 10981483)

**2.11.1.4.0-2.0-2**  (ID: `10981493`)

PsTrace Pcmd Timing Advance Indication
[Before CNI-122508-B]  
When the feature is activated, after receiving Timing Advance PCMD Record indication from RT L2-PS, NRT L2-PS should prepare PsTrace\_PcmdTimingAdvanceInd and send it to the drain address.

PsTrace\_PcmdTimingAdvanceInd shall be triggered only in order to report Periodic Timing Advance.

Rationale: As specified in 5G\_UP\_3598\_replaced([11400765](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ugGlQPvkEe-AqvopbP1qhQ)), PCMD Trace Report Framework shall generate messages only for periodic U-Plane PCMD records. Tracing of event based PCMD records is not supported.  
[End CNI-122508-B]
