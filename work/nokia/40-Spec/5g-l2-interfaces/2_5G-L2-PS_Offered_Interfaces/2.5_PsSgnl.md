## 2.5.2 Procedures

### 2.5.2.1 Pcch Data Send Request

#### 2.5.2.1.0-1 Pcch Data Send Request Overview (ID: 10978558)

**2.5.2.1.0-1.0-1**  (ID: `12815329`)

Pcch Data Send Request
Messages:

PsSgnl\_PcchDataSendReq

Purpose:

To request sending of DL PCCH data (RRC paging).

Precondition:

Address distribution is done.

Addressing:

The request is sent to the PsSgnl address.

There is no response related to the request.

Relevant UP MSCs:

5G\_L1L2\_FLOW\_245\_replaced([11194184](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BfohePvREe-AqvopbP1qhQ)): DL data on PCCH (RLC TM): RRC paging

5G\_L1L2\_FLOW\_247\_replaced([11194202](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BfohevvREe-AqvopbP1qhQ)):DL data on PCCH: system information modification

#### 2.5.2.1.0-2 Pcch Data Send Request Description (ID: 10978563)

**2.5.2.1.0-2.0-1**  (ID: `12815331`)

Pcch Data Send Request
Upon reception of the request, 5G-L2-PS shall initiate sending of a RRC paging and/or notification of system information change on DL PCCH.

---

#### 2.5.2.1.0-3 Pcch Data Send Request Message validation (ID: 10978567)

**2.5.2.1.0-3.0-1**  (ID: `12815332`)

Pcch Data Send Request
Upon reception of the PsSgnl\_PcchDataSendReq message, 5G-L2-PS shall check the validity of the request message.

For exception handling, see 5G\_L2\_IF\_575\_replaced([10978572](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2VUnKfu6Ee-AqvopbP1qhQ)).

---

#### 2.5.2.1.0-4 Pcch Data Send Request Exceptions (ID: 10978572)

**2.5.2.1.0-4.0-1**  (ID: `12815333`)

Pcch Data Send Request
Expected Exceptions:

[5GC002184-E]

Generic exception handling rules can be found in the chapter 5G\_UP\_1795\_replaced([11374912](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueKrmfvkEe-AqvopbP1qhQ)).

5G-L2-PS shall discard the information for a cell if nrCellIdentity IE is not known (cell does not exist)

5G-L2-PS shall discard the message if the request message is not valid.

Unexpected Exceptions:

In case of an unexpected exception, 5G-L2-PS shall discard the message.

Error is logged for both expected and unexpected exception.

[End 5GC002184-E]
