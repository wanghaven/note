#### 2.6.2.1.0-2 PsTest Info Request Description (ID: 10978645)

**2.6.2.1.0-2.0-2**  (ID: `10978654`)

PsTest Info Request
[Before CNI-109596]

Upon reception of the request, 5G-L2-PS shall save smDataUePdcchIndAddress that shall be used for sending SmDataUe\_PdcchInd message carrying the scheduled DCI.

5GUE-L2-SM sends periodicity to 5G-L2-PS a dummy message to maintain the RRC MAC learning table on EFI2 port with 5GUE-L2-SM  mac address. 5G-L2-PS shall also allocate a EQID with a fake local queue Id to delete this incoming message.

EQID client address smDataUePdcchIndAddress, provided by 5GUE-L2-SM to 5G-L2-PS, shall correspond to the event queue attached to backplane PEP of the core. Corresponding "EM queue create" API pointing to backplane PEP ( First\_Device\_id = default) shall be used when EM queue is created (OAM should configure RRC switch such that EFI2 to be in the backplane VLAN).

When created EQID client address smDataUePdcchIndAddress, on ABIL, 5GUE-L2-SM shall use API "EM queue create" with second\_device\_id parameter pointing now to backplane RRC.

L2-PS shall send both PsTest\_SubCellGroupInfoResp and PsTest\_SubcellGroupInfoBfcmBasedResp after reception of PsTest\_SubCellGroupInfoReq

Upon reception of empty message PsTest\_CellsMappingReq, 5G-L2-PS instance shall reply with information of cells handled by that instance.

[End CNI-109596]

[CNI-109596]

EQID client address smDataUePdcchIndAddress is no more used and has been removed.

[End CNI-109596]

#### 2.6.2.3.0-2 PsTest PDCCH Ind Address Description: (ID: 10978752)

**2.6.2.3.0-2.0-3**  (ID: `10978766`)

PsTest PDCCH Ind Address
[Before CNI-109596]

Upon reception of the request, 5G-L2-PS shall save smDataUePdcchIndAddress that shall be used for sending SmDataUe\_PdcchInd message carrying the scheduled DCI.

5GUE-L2-SM sends periodicity to 5G-L2-PS a dummy message to maintain the RRC MAC learning table on EFI2 port with 5GUE-L2-SM  mac address. 5G-L2-PS shall also allocate a EQID with a fake local queue Id to delete this incoming message. 5G-L2-PS shall used EQID associated to the PsTest interface to reveive the message, but shall not do any processing

When created EQID client address smDataUePdcchIndAddress, on ABIL, 5GUE-L2-SM shall use API "EM queue create" with second\_device\_id parameter pointing now to backplane RRC.

[End CNI-109596]

[CNI-109596]

EQID client address smDataUePdcchIndAddress is no more used and has been removed.

[End CNI-109596]
