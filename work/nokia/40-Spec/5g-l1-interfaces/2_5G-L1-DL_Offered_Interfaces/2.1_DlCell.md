#### 2.1.1.1.3 DL Cell Parameter Reconfiguration

##### 2.1.1.1.3.1 Procedure

**2.1.1.1.3.1.0-3**  (ID: `11512243`)

Procedure
Expected Exceptions:

For Loner, Loki and Thor BB:

Reply a DlCell\_ParameterReconfigurationResp for each DlCell\_ParameterReconfigurationReq received:

- set the IE status = EStatus5G\_Ok (0) if online modification of ssBlockPower is successful. Otherwise set the IE status = EStatus5G\_NotOk (1).

- set the IE cause and IE diagnosticInformation according to 5G\_L1\_IF\_14795\_replaced([11512188](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FRnXzfvxEe-AqvopbP1qhQ)).

Unexpected Exceptions:

In case of unexpected exception, no response message is returned.
