## 15.4.2 Description

### 15.4.2.0-1 Description (1) UL CG type2 non-GBR load report (ID: 11208436)

**15.4.2.0-1.0-1**  (ID: `11208445`)

Description
L2-PS report UL CG type2 non-GBR PRB and TBS load to CP-RT through message PsCell\_UlNonGbrLoadInd defined in 5G\_L2\_IF\_4401\_replaced([10973764](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2U3UKvu6Ee-AqvopbP1qhQ)) periodically. Report period and load calculation is specified in 5G\_UP\_ALG\_15450\_replaced([11201437](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kwWAnvvSEe-AqvopbP1qhQ)).

---

### 15.4.2.0-2 Description (2) UL CG activation control (ID: 11208455)

**15.4.2.0-2.0-1**  (ID: `11208463`)

Description
CP-RT requests L2-PS to suspend GBR or non-GBR UL CG type2 activation through message PsCell\_UlCgType2ActivationControlReq defined in 5G\_L2\_IF\_4617\_replaced([10973857](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2U37PPu6Ee-AqvopbP1qhQ)) when necessary.  L2-PS need to feedback local GBR and non-GBR UL CG type2 activation control state to CP-RT through message PsCell\_UlCgType2ActivationControlResp defined in 5G\_L2\_IF\_4623\_replaced([10973884](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2U4iQfu6Ee-AqvopbP1qhQ)) after state update.

---

### 15.4.2.0-3 Description (3) CP-RT request L2-PS to reduce UL non-GBR CG type2 PRB load (ID: 11208472)

**15.4.2.0-3.0-1**  (ID: `11208479`)

Description
CP-RT may requests L2-PS to reduce non-GBR UL CG type2 PRB load when new DRB setup or modify happened. The requested PRB load reduction value is defined as percentage value 'percentToRecude' per DRB in message PsUser\_BearerSetupReq or PsUser\_UserSetupReq or PsUser\_DrbModifyReq refer to 5G\_L2\_IF\_397\_replaced([10974876](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2U-o4vu6Ee-AqvopbP1qhQ)) and 5G\_L2\_IF\_4782\_replaced([10977913](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2VPHn_u6Ee-AqvopbP1qhQ)).

L2-PS calculate final PRB load reduce value according to 'percentToReduce' value as specified in 5G\_UP\_ALG\_17873\_replaced([11173250](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kuSLFfvSEe-AqvopbP1qhQ)) and select non-GBR UL CG type2 UEs to deactivate CG as specified in 5G\_L2\_12008\_replaced([11176718](https://dn-prod.ext.net.nokia.com/rm/resources/BI_262Ms_vTEe-AqvopbP1qhQ)).
