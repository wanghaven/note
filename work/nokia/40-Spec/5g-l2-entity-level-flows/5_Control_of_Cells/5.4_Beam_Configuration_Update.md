**5.4.0-2.0-9.0-2**  (ID: `12701006`)

Beam Configuration Update
When [Before CB014301-A] cell's down tilt [CB014301-A] beam set parameters [End CB014301-A] change, SSB/refine beam's degree of angle may change so CPRT informs the change to L2-PS by sending PsCell\_CellSsbRefineBeamDoaInd as [10974196: Cell SSB/refine Beam DOA Indication](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2U6Xefu6Ee-AqvopbP1qhQ?oslc_config.context=https%3A%2F%2Fdn-prod.ext.net.nokia.com%2Fgc%2Fconfiguration%2F9).When receiving this information, L2-PS shall update SSB/refine beam's degree of angle. The sequence of the message PsCell\_CellSsbRefineBeamDoaInd takes no specific setting for L2-PS, i.e. it can be before or after  a PsCell\_CellReconfigurationReq/resp.

During cell setup the sCell\_CellSsbRefineBeamDoaInd message ([11057628](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BCOMnv_KEe-D3seUwaI0tg?vvc.configuration=https://dn-prod.ext.net.nokia.com/rm/cm/stream/_NS8shu3xEe-rL47x_rCCmw)) is delivered towards L2-PS, that includes ssbRefineBeamDoA structure. It is calculated in [11288965: Pre-defined beam set configuration](https://dn-prod.ext.net.nokia.com/rm/resources/BI_Tp-J8fvcEe-AqvopbP1qhQ?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9) that is impacted by CB014005-A and CB014301-A scenarios.
