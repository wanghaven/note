[image-0][image\_desc]The image is a flowchart illustrating the RIM RS indication flow when mitigation is set to DL or UL+DL in a 5G network. The flowchart depicts a sequence of events and state changes triggered by the detection or disappearance of RIM RS (Radio Interface Monitoring Radio Signal) and its impact on fault reporting and mitigation.
The flowchart starts with an initial state where 5G-L2-PS detects Remote Interference (RI) and changes the state of Remote Interference detection from State1 to State2. This leads to a fault being reported (Fault 7527). The flow then branches based on the detection of RIM RS1 and RIM RS2, and the number of detected set IDs relative to a mitigation threshold.
Key events and states include:
- Detection of RIM RS1 and RIM RS2 in State 2.
- Changes in the number of detected set IDs.
- Disappearance of RIM RS1 or RIM RS2.
- Detection of new RIM RS1 signals.
- The state of DL mitigation (stopped or continued).
- Fault reporting, updating, cancellation, and triggering.
The flowchart uses arrows to indicate the flow of control and events. Each step is labeled with a "Step" number and a description of the action or condition. The right side of the flowchart shows interactions with a "5G-CP-RT" entity, indicating fault reporting and updates.
The text within the flowchart describes the following:
**Initial State:**
- **5G-L2-PS:**
- `actNrRim = "framework1"`
- 5G-L2-PS detects Rl and changes the state of the Remote Interference detection from State1 to State2(2a = true, 2b = false, 2c = false)
- see 5G\_UP\_ALG\_9180 for details
**Flowchart Steps:**
\* **(Step 0)PsCell\_Remotelnterferenceind(remoteInterferenceDetected=true, rimRsDetectOngoing=true, rimRsDetected=notDetected)**
\* **RIM RS1 has been detected in State 2**
- and state changes to State 2(2a= true, 2b=true, 2c = false)
- Number of detected setids < numDetectedSetIdForMitigation
- **(Step 1)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=true, rimRsDetectOngoing=true, rimRsDetected=Type1, dlPdschMuting={NoMuting), dlPdcchMuting = False, rimRs1DetectedSetlds={ID1})**
- **Fault 7527 is reported**
\* **RIM RS2 has been detected in State 2**
- and state changes to State 2(2a= true, 2b=true, 2c = true)
- Number of detected setids < numDetectedSetIdForMitigation
- **(Step 1a)PsCell\_Remotelnterferencelnd (remoteInterferenceDetected=true, rimRsDetectOngoing true, rimRsDetected=Type1AndType2, dlPdschMuting={NoMuting), dlPdcchMuting = False, rimRs1DetectedSetlds={ID1), rimRs2DetectedSetlds={ID1})**
- **Fault 7527 is updated**
\* **New RIM RS1 signal is detected with new setld**
- Number of detected setlds >= numDetectedSetIdForMitigation
- RIM RS2 transmission starts
- **(Step 2)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=true, rimRsDetectOngoing=true, rimRsDetected=Type1AndType2, dlPdschMuting={See 5G\_L2\_IF\_2287 for possible values), dlPdcchMuting = False, rimRs1DetectedSetIds={ID1, ID2), rimRs2DetectedSetlds={ID1))**
\* **One RIM RS1 disappears in current transmission period.**
- Number of detected setids < numDetectedSetIdForMitigation
- DL mitigation is stopped
- **(Step 3)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=true, rimRsDetectOngoing=true, rimRsDetected=Type1AndType2, dlPdschMuting={NoMuting), dlPdcchMuting = False, rimRs1DetectedSetlds={ID1), rimRs2DetectedSetlds={ID1))**
\* **Second RIM RS1 disappears in current transmission period, state changes to State 2(2a= true, 2b=false, 2c = true)**
- RIM RS2 continues as RIM RS2 is still detected
- **(Step 4)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=true, rimRsDetectOngoing=true, rimRsDetected=Type2, dlPdschMuting={NoMuting), dlPdcchMuting = False, rimRs1DetectedSetIds={}, rimRs2DetectedSetIds={ID1))**
\* **RIM RS2 disappears in current transmission period, state changes to State 2(2a= true, 2b=false, 2c = false)**
- RIM RS2 transmission stops
- **Fault 7527 is cancelled**
\* **RIM RS1 has been detected again in State 2(2a= true, 2b=false)**
- and state changes to State 2(2a= true, 2b=true),
- RIM RS2 transmission starts,
- Number of detected setids < num DetectedSetIdForMitigation
- **(Step 5)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=true, rimRsDetectOngoing=true, rimRsDetected=Type1, dlPdschMuting={NoMuting), dlPdcchMuting = False, rimRs1DetectedSetlds={ID1})**
- **Fault 7527 is triggered due to RIM RS1 detection**
\* **RI disappears but still at least one RIM RS1 has confirmed detection.**
- Detection of new RIM RS1 is stopped.
- Only revalidation of detected Setlds is maintained.
- **(Step 6)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=false, rimRsDetectOngoing = false, rimRsDetected=Type1, dlPdschMuting={NoMuting), dlPdcchMuting = False, rimRs1DetectedSetlds={ID1})**
\* **None of revalidated Setld is left,**
- RIM RS1 monitoring stops,
- RIM RS2 transmission stops
- **(Step 7)PsCell\_RemoteInterferencelnd(remoteInterferenceDetected=false, rimRsDetectOngoing = false, rimRsDetected=notDetected, dlPdschMuting={NoMuting), dlPdcchMuting = False)**
