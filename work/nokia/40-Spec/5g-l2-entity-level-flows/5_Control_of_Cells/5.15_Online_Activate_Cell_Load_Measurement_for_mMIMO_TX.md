**5.15.0-2.0-1.0-1**  (ID: `12811318`)

Online Activate Cell Load Measurement for mMIMO TX Power Saving
[Before PR867026] For FR1 TDD [End PR867026] To configure online NR load-based mMIMO TX power saving, 5G-L2-PS receives PsCell\_CellSetupReq with:

* [PR867026] cellTechnology = TDD [End PR867026]
* actBeamforming = TRUE
* isCellMassiveMimo=TRUE
* actExternalAntennaBF = FALSE
* actNrLBmMimoRTXPowerSaving= FALSE

with pattern ids for power saving filled depending on beamforming method is used:

* For SSB/refined (CSI-RS) with bfSetup.mMimoPSBeamWeightVector.patternId AND
* For PMI with startCsirsPmiPatternIdForNrLBmMimoRTXPowerSaving AND/OR
* For SRS based DFT with:
* bfSetup.dftBeamSetList.dftBeamSetId and
* bfSetup.fr1Fr2Combined.fr1Freq.dftBeamSetList.dftBeamStartPatternId

which are used in pattern id power saving calculation algorithm. 5G-L2-PS shall store all values of pattern ids for power saving.

See more: [12763413](https://dn-prod.ext.net.nokia.com/rm/resources/BI_m61BxIPzEfCOGMMH109ICw?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9)

* actEirpControl= TRUE from the last stored value (PsCell\_CellSetupReq OR PsCell\_CellReconfigurationReq)
* eirpControlForSrsDftBeam if:
* actDlSrsBm = TRUE in PsCell\_CellSetupReq AND
* actEirpControl= TRUE from the last stored value  (PsCell\_CellSetupReq OR PsCell\_CellReconfigurationReq)
