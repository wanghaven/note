# 8.3 IFFT

**8.3.0-1**  (ID: `13257062`)

IFFT
5G-L1-DL (FHE-CPRI-GW) shall support to apply IFFT to the I/Q frequency samples for each stream for FR1 FDD.

Using parameters bandwidth and iqSampleCount, provided in API\_BBB\_CREATE\_EAXC\_AXC\_ROUTING\_REQ\_MSG (with direction "EBbbEaxcDirection\_Downlink"), dlBandwidth\_PRBs and iFFT size can be obtained in [13255777: DL Bandwidth configurations](https://dn-prod.ext.net.nokia.com/rm/resources/BI_UhS7E8kSEfCYBIz6acpPHw?oslc_config.context=https%3A%2F%2Fdn-prod.ext.net.nokia.com%2Fgc%2Fconfiguration%2F9). There parameters are used to derive the number of zero padding subcarriers at the input of the iFFT, to be added symmetrically on both sides of the useful bandwidth.

The number of zero padding subcarriers equals to (NIFFTsize- 12 x dlBandwidth\_PRBs), with NIFFTsize- stands for iFFT size.
