# 8.1 DL Routes Configuration

## 8.1.0-1 DL Routes Configuration 5G-L1-DL (FHE-CPRI-GW) shall configure the DL routes with the parameters upon reception of API_BBB_CREATE_EAXC_AXC_ROUTING_REQ_MSG message (with direction is "EBbbEaxcDirection_Downlink") from RUProxy. See 12981526: L1 Configuration Management for the details of L1 configuration management (ID: 13345767)
