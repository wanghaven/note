# 9.5 FFT

**9.5.0-1**  (ID: `13346435`)

FFT
5G-L1-UL (FHE-CPRI-GW) shall support to apply FFT to the I/Q frequency samples for each stream for FR1 FDD.

Using parameters bandwidth and iqSampleCount, provided in API\_BBB\_CREATE\_EAXC\_AXC\_ROUTING\_REQ\_MSG (with direction "EBbbEaxcDirection\_Uplink"), ulBandwidth\_PRBs and FFT size can be obtained in [13346434](https://dn-prod.ext.net.nokia.com/rm/resources/BI_CgjB0uTyEfCmxLW1M7HMrQ?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9)[.](https://dn-prod.ext.net.nokia.com/rm/resources/BI_dQFbdcmnEfCYBIz6acpPHw?oslc_config.context=https://dn-prod.ext.net.nokia.com/gc/configuration/9) There parameters are used to derive the number of zero padding subcarriers at the output of the FFT, to be removed symmetrically on both sides of the useful bandwidth.

The number of zero padding subcarriers equals to (NFFTsize- 12 x ulBandwidth\_PRBs), with NFFTsize- stands for FFT size.
