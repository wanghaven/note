## 11.8.2 Downlink Cell Capacity

### 11.8.2.0-15 Downlink Cell Capacity 5G-L1-HI shall support following configuration: DL subcells inside same L1 instance shall have same DlCell_SetupReq -&gt; dlSubcellType. (ID: 11477407)

## 11.8.3 Uplink Cell Capacity

### 11.8.3.0-15 Uplink Cell Capacity 5G-L1-HI shall support following configuration: UL subcells inside same L1 instance shall have same UlCell_SetupReq -&gt; ulSubcellType. (ID: 11477474)

## 11.8.4 Downlink Channel Capacity

### 11.8.4.1 PDCCH

#### 11.8.4.1.0-4 PDCCH 5G-L1-HI shall support following maximum number of DCIs per subcell per slot: 32 (16UL + 16DL). (ID: 11477500)

### 11.8.4.2 PDSCH

#### 11.8.4.2.0-10 PDSCH 5G-L1-HI shall support following number of UEs scheduled per subcell per slot: 16. (ID: 11477556)

### 11.8.4.3 CSI-RS

#### 11.8.4.3.0-7 CSI-RS 5G-L1-HI shall support up to 8 CSI-RS resources per subcell per slot for CSI-RS refinement mode. (ID: 11477596)

## 11.8.5 Uplink Channel Capacity

### 11.8.5.1 PUCCH

#### 11.8.5.1.0-10 PUCCH 5G-L1-HI shall support up to 22 (RM or Polar) PUCCH F3 resources per subcell per slot in real-time processing. (ID: 11477651)

### 11.8.5.2 PUSCH

#### 11.8.5.2.0-16 PUSCH 5G-L1-HI shall support following number of HARQ-A/N resources scheduled on PUSCH per subcell per slot: 16. (ID: 11477721)

### 11.8.5.3 SRS

#### 11.8.5.3.1 General

##### 11.8.5.3.1.0-1 General 5G-L1-HI shall use separate resources for SRS_SU-MIMO and SRS_BM. (ID: 11477733)

#### 11.8.5.3.2 SRS for SU-MIMO

##### 11.8.5.3.2.0-3 SRS for SU-MIMO 5G-L1-HI shall use separate SRS_SU-MIMO resources for each UL subcell. (ID: 11477749)

#### 11.8.5.3.3 SRS for Beam Management

### 11.8.5.4 PRACH

#### 11.8.5.4.1 Format 0

##### 11.8.5.4.1.0-2 Format 0 5G-L1-HI shall support PRACH configuration index (PCI) = [0..7, 9, 12, 14, 17] for TDD, based 3GPP specification (v.16.4.0). (ID: 11477769)

#### 11.8.5.4.2 Format A2

##### 11.8.5.4.2.0-1 Format A2 5G-L1-HI shall support PRACH configuration index (PCI) = [87…90, 92, 93, 94, 96, 97, 98, 100, 101, 102, 104] for TDD, based on 3GPP specification (v.16.4.0). (ID: 11477777)

#### 11.8.5.4.3 Format B4

##### 11.8.5.4.3.0-1 Format B4 5G-L1-HI shall support PRACH configuration index (PCI) = [145…152, 156..160, 161,162] for TDD, based on 3GPP specification (v.16.4.0). (ID: 11477784)

#### 11.8.5.4.4 Format C2

##### 11.8.5.4.4.0-1 Format C2 5G-L1-HI shall support PRACH configuration index (PCI) = [189…192, 194..198, 201,202,203,205, 206] for TDD, based on 3GPP specification (v.16.4.0). (ID: 11477791)
