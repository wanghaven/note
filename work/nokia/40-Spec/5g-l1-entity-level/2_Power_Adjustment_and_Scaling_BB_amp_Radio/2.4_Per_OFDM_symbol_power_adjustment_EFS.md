# 2.4 Per OFDM symbol power adjustment EFS

## 2.4.1 For Loner only

**2.4.1.0-2**  (ID: `11418290`)

For Loner only
Message DlData\_SlotTypeReq, sent by 5G-L2-PS to 5G-L1-DL, contains a parameter pwrReductionPerSymb\_dB, which is an array of fourteen unsigned integers on 8 bits (u8 format). Each of these integers, e.g. pwrReductionPerSymb\_dB[i], gives the power reduction in dB that is to be applied to the corresponding OFDM symbol i in addition to the per channel/signal power scalings described in previous sections. This per OFDM symbol power reduction is implemented before IFFT, in the RE mapping stage (for both the CPRI and eCPRI cases).

The way the power reduction is encoded with the above mentioned u8 integers is specified in [11512413: DlData\_SlotTypeReq message content](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FRpM9vvxEe-AqvopbP1qhQ) and is reminded in the following table, which in addition provides the corresponding linear scaling factors to be applied.

|  |  |  |
| --- | --- | --- |
| pwrReductionPerSymb\_dB[i] | Power reduction to be applied to OFDM symbol i | Linear scaling factor |
| 0 | 0 dB (no power reduction) | 1 |
| 1 | 1 dB | 10-1/20 ≈ 0.89125 |
| 2 | 2 dB | 10-2/20 ≈ 0.79433 |
| … | … | … |
| x | x dB | 10-X/20 |
| … | … | … |
| 50 | 50 dB | 10-50/20 ≈ 0.00316 |
| From 51 to 255 | Mute OFDM symbol completely | 0 |

In downlink, parameter pwrReductionPerSymb\_dB in the DlData\_SlotTypeReq message does not apply to the CSI-RS transmissions. Instead, the DlData\_CsiRsSendReq message contains, for each CSI-RS resource within the message, a parameter pwrReductionPerCsiRsResource\_dB that specifies a power reduction to be applied to the OFDM symbol(s) of the CSI-RS resource in the slot. See 5G\_L2\_569 for an explanation of why this new parameter is needed. The coding of the parameter is the same as the one used for pwrReductionPerSymb[i] and shown in above table. This parameter will be used in two different ways:

 if the trsInfo parameter for the CSI-RS resource in the DlData\_CsiRsSendReq message is set to DISABLED, the CSI-RS resource is considered to span only one OFDM symbol and the pwrReductionPerCsiRsResource\_dB parameter is a power reduction to be applied to that OFDM symbol only;

 if the trsInfo parameter for the CSI-RS resource in the DlData\_CsiRsSendReq message is set to ENABLED, the CSI-RS resource is considered to span two OFDM symbols, namely OFDM symbols l0 and l0 + 4, where l0 is the OFDM symbol number provided by parameter startSymbol of the CSI-RS resource.

**[CNI-81900]L2-PS only sends SlotTypeReq if it has non-zero powerReduction value for at least one symbol in the slot. L1 shall apply the non-zero power reduction values in the scheduled slot and symbols, while for other slots without slotTypeReq, L1 shall apply zero power reduction value.**[End CNI-81900]\*\*\*\*
