# 3.5 L2-L1 Message Timing

## 3.5.1 Discard message out of reption window

**3.5.1.0-1**  (ID: `11415118`)

Discard message out of reption window
L1 shall discard the messages out of the L1 reception window.

[CB009514-A] This requirement shall be also applied to eCPRI 7-2e. [CB009514-A]

[CNI-122313]

RINLINE1 support is terminated in 24R1, requirement is not valid for RINLINE1 starting from 24R2.

[End CNI-122313]

## 3.5.4 Log behavior when message out of reception window

### 3.5.4.0-1 Log behavior when message out of reception window When a L2-L1 request message is received out of reception window: L1 shall not log error. L1 should log warning or use other means to keep tracking of it. Rationale: Error log will cause test case failure which is not expected.See PR612814. (ID: 11415852)
