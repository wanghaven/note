## 6.7.4 rdSwitchToLowerPrecision

**6.7.4.0-1**  (ID: `11468910`)

rdSwitchToLowerPrecision
rdSwitchToLowerPrecision:

This parameter indicates higher precision data type or lower precision data.

* 0: higher precision
* Use higher precision data type (vc\_acc\_t), which has 15 bits mantissa.
* 1: lower precision
* User lower precision data type(vc\_t), which has only 7 bits mantissa.

Default value: 0

Online modifiable: Yes

Note: This parameter is applied on Thor RU.

Specified in: [11468159](https://dn-prod.ext.net.nokia.com/rm/resources/BI_fBoEF_vvEe-AqvopbP1qhQ)

Trigger for deletion: Never.
