#### 6.4.4.1.2 Control resource set definition2

##### 6.4.4.1.2.1 Thor RU

**6.4.4.1.2.1.0-1**  (ID: `11468259`)

Thor RU
For a PDCCH resource set (CORESET + first symbol), a combination is defined with the following attributes: {first symbol, number of symbols, start Common Resource Block (CRB, see 38.211 §4.4.4.3) & number of RBs}.

These attributes are derived from the *DlDataFH\_PdcchSendReq message (see 5G\_L1\_IF\_15119\_replaced([11516750](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FSGf8fvxEe-AqvopbP1qhQ))) as follow:*

* The first symbol corresponds to the parameter *startSymbolNumber, and the number of symbols corresponds to the parameter* numOfSymbols, refer to 5G\_L1\_4348.\*\*
* **The start and last CRB and the number of RBs are computed from the parameters *coresetFreqDomain (64-bit bitmap) and \*coresetFreqDomainRbShift:***\*
* start CRB = 6*(63 - first MSB bit in* coresetFreqDomain bitmap) + *coresetFreqDomainRbShift*\*
* last CRB = 6*(64 - last MSB bit in* coresetFreqDomain bitmap) - 1 + *coresetFreqDomainRbShift*\*
* number of RBs = 6 *number of '1' in* coresetFreqDomain bitmap\*
