## 11.1.2 Cell Configuration

### 11.1.2.1 General

#### 11.1.2.1.0-9 General 5G-L1-Hi shall accept DlCell_SetupReq-&gt;conformanceTestMode = {true, false}, but the value should not affect 5G-L1-Hi behavior. (ID: 11470905)

### 11.1.2.2 Subcell types

#### 11.1.2.2.0-4 Subcell types 5G-L1-HI shall support C4 DL subcell type (DlCell_SetupReq-&gt;dlSubcellType = C4). (ID: 11470928)

### 11.1.2.3 Bandwidth

#### 11.1.2.3.0-9 Bandwidth 5G-L1-HI shall support 100 MHz DL subcell bandwidth (DlCell_SetupReq-&gt;dlBandwidth = 273 with DlCell_SetupReq-&gt;scs = khz_30 ). (ID: 11470975)

### 11.1.2.4 DL MIMO Mode

#### 11.1.2.4.0-7 DL MIMO Mode 5G-L1-HI shall support DlCell_SetupReq-&gt; dlMimoMode = {CL_8x4_MIMO = 4} with DlCell_SetupReq-&gt; dlSubcellType = C4 and  DlCell_SetupReq-&gt;actBeamforming = true. (ID: 11471013)

### 11.1.2.5 Beamforming

#### 11.1.2.5.0-2 Beamforming 5G-L1-HI shall support DL FD (Freqeuncy Dependent) beamforming when DlCell_SetupReq-&gt;actBeamforming = {true} and DlCell_SetupReq-&gt; dlEcpriFdBeamforming = {true}. (ID: 11471029)

### 11.1.3 SSB/PBCH

#### 11.1.3.0-27 SSB/PBCH 5G-L1-HI shall support following configuration: DlCell_SetupReq-&gt;dlSubcellType = {C4} and DlCell_SetupReq-&gt;dlMimoMode = {CL_4x4_or_4x2_MIMO = 1, OL_4x4_or_4x2_MIMO = 3} and DlCell_SetupReq-&gt; actBeamforming = {false} and DlData_SsBlockSendReq -&gt; numCeAxcIndex = {4} and DlData_SsBlockSendReq -&gt; ceAxCIndex = (0, 1, 2, 3) (ID: 11471165)

### 11.1.4.2 DCI configuration

#### 11.1.4.2.0-5 DCI configuration 5G-L1-HI shall adjust PDCCH transmission power according to DlData_PdcchSendReq-&gt;pdcchDciTransmitPower. (ID: 11471314)

## 11.1.5 PDSCH

### 11.1.5.1 Signal Generation

#### 11.1.5.1.0-1 Signal Generation 5G-L1-HI shall support signal generation (along with associated DMRS) upon reception of DlData_PdschSendReq and DlData_PdschPayloadTbSendReq messages. (ID: 11471437)

### 11.1.5.2 Resource Allocation

#### 11.1.5.2.0-27 Resource Allocation 5G-L1-HI shall withdraw DlData_PdschSendReq-&gt;numCsiRsMappingRes REs from PDSCH PRBs that overlap with PRBs scheduled for CSI-RS for tracking if DlData_PdschSendReq-&gt;isCsirsRateMatching = {true}. (ID: 11471521)

#### 11.1.5.4.0-8 Modulation and Coding 5G-L1-HI shall support 256-QAM table for all MCSs in 3GPP 38.214 (Table 5.1.3.1-2, v.15.3.0). (ID: 11472437)

### 11.1.5.5 DMRS

**11.1.5.5.0-14**  (ID: `11472494`)

DMRS
5G-L1-Hi shall support DlData\_PdschSendReq->offsetRbDmrs.

Value equal to {0} means that the reference point for DMRS sequence generation is the first PRB index of the bandwidth (equal to {0}).

Value not equal to {0} means the the reference point for DMRS sequence generation is the first PRB index of the initial bandwidth part (or, in case of SIB1 message, lowest PRB of CORESET0) which is equal to DlData\_PdschSendReq->offsetRbDmrs.

## 11.1.6 CSI-RS

### 11.1.6.1 General

#### 11.1.6.1.0-3 General 5G-L1-HI shall support transmission of CSI-RS for refinement (beam management) when DlData_CsiRsSendReq -&gt; trsInfo = DISABLED. (ID: 11472533)

### 11.1.6.2 Signal Generation

#### 11.1.6.2.0-2 Signal Generation 5G-L1-HI shall use DlData_CsiRsSendReq -&gt; csiRsScramblingSequenceInt in CSI-RS signal generation as a value of scramblingID parameter specified in 3GPP 38.214 (V15.6.0), chapter 5.2.2.3.1. (ID: 11472545)

### 11.1.6.3 Resource Allocation

#### 11.1.6.3.1 CSI-RS Configuration Index

##### 11.1.6.3.1.0-6 CSI-RS Configuration Index 5G-L1-HI shall support DlData_CsiRsSendReq -&gt; csiRsConfig = {2} along with DlData_CsiRsSendReq -&gt; density = {one} for CSI-RS refinement mode. (ID: 11472574)

#### 11.1.6.3.2 Start Symbol

##### 11.1.6.3.2.0-6 Start Symbol 5G-L1-HI shall support DlData_CsiRsSendReq -&gt; startSymbol = {[2..5], [8..11]} for CSI-RS refinement mode. (ID: 11472601)

#### 11.1.6.3.3 Start PRB

##### 11.1.6.3.3.0-4 Start PRB 5G-L1-HI shall support DlData_CsiRsSendReq -&gt; startPrb equal to multiples of 4 for CSI-RS refinement mode, so that CSI-RS is located either below or above SS/PBCH blocks. (ID: 11472621)

#### 11.1.6.3.4 Number of PRBs

##### 11.1.6.3.4.0-4 Number of PRBs 5G-L1-HI shall support DlData_CsiRsSendReq -&gt; numOfPrb equal to all multiples of 4 which fit to either below or above SS/PBCH block for CSI-RS refinement mode. (ID: 11472642)

#### 11.1.6.3.5 Frequency Domain Allocation Bitmap

##### 11.1.6.3.5.0-5 Frequency Domain Allocation Bitmap 5G-L1-HI shall support DlData_CsiRsSendReq -&gt; freqDomainAllocationKi where two bits from index set {0, 2, 4, 6, 8, 10} should be set to '1' (counting from LSB) for csiRsConfig = 8. (ID: 11472667)
