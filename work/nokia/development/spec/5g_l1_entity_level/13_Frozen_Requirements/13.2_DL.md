# 13.2 DL

## 13.2.1 DL Common

### 13.2.1.1 Cell bandwidth and configuration

#### 13.2.1.1.0-4 Cell bandwidth and configuration Requirement is incorrect/invalid. L1 FPGA shall support cell reconfiguration in runtime without BB capacity board reset, so that It does not impact existing, active resources during configuration changes. (ID: 11489663)

### 13.2.1.2 subCell configuration

**13.2.1.2.0-1**  (ID: `11489674`)

subCell configuration

[Before CNI-122313]

* For TDD FR1 with eCPRI:

- 0 "CL 2x2 MIMO"

- 1 "CL 4x4 or 4x2 MIMO"

[CB009881-CG]

[End CB009881-CG]

## 13.2.3 CSI-RS

### 13.2.3.1 CSI-RS configuration

**13.2.3.1.0-1**  (ID: `11489720`)

CSI-RS configuration
L1 shall support to transmit the CSI-RS for refinement for FR1 TDD eCPRI with the following configuration in RINLINE1:

[Before CNI-122313]

[CB009881-C] Parity from 5GC000534

• In case of csiRsConfig=1:

- density = 3

- k0 = {0,1,2,3}

• In case of csiRsConfig = 2:

- density = 1

- k0 = {0...11}

• trsInfo = DISABLED

• startSymbol = {2,3,4,5,8,9,10,11}

• startPrb:may be located either below or above SS/PBCH blocks, startPRB is hence either at 0 or the first PRB above SS/PBCH block transmission.

• numOfPrb:all multiples of 4 which fit to either below or above SS/PBCH block allocation

[End CB009881-C]

PRB Blanking validation

[Before 03235504]

* 5G-L1-DL shall check *numOfPrb +* startPrb against *dlBandwidth. If* numOfPrb + *startPrb >* dlBandwidth, L1 shall transmit CSI-RS on PRB indices *startPrb through* dlBandwidth-1.***\****\*

[End 03235504]
