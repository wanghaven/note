# 9 5G-L1-UL (FHE-CPRI-GW)

**9.0-1**  (ID: `13133240`)

5G-L1-UL (FHE-CPRI-GW)
5G-L1-UL (FHE-CPRI-GW) part is responsible for

Standard ingress rout calculation:

* IFFT calculation
* Cyclic prefix removal

PRACH ingress rout calculation:

* Static PRACH calculations

5G-L1-UL (FHE-CPRI-GW) is configured by FHE messages.

UL is responsible for [Egress direction](https://dn-prod.ext.net.nokia.com/rm/resources/TX_nOb-5sVdEfCnuMJFWtMTpQ) (CPRI RU→GW→eCPRI DU)

IngressPort for UL means Input port of AxC

EgressPort for UL means output port of eAxC

UL timing budget / Synchronization

Pass through layer

Simple translations

Complex translation
