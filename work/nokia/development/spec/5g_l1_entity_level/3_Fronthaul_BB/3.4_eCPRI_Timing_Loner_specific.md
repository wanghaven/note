## 3.4.3 Destination MAC address configuration in eFramer

### 3.4.3.0-1 Destination MAC address configuration in eFramer eFramer shall support destination MAC address per virtual stream, configured by L1SW. (ID: 11415072)

## 3.4.4 Destination MAC address re-configuration in eFramer

### 3.4.4.0-1 Destination MAC address re-configuration in eFramer L1SW shall configure destination MAC addresses and physical Fronthaul port per c_eAxC in the eFramer based on the contents of API2_L1SW_CONFIGURE_TRANSPORT_REQ_MSG. L1SW shall support re-configuration of destination MAC addresses in runtime during cell deletion/addition procedure. (ID: 11415077)

## 3.4.5 VLAN ID configuration in eFramer

### 3.4.5.0-1 VLAN ID configuration in eFramer eFramer shall support configuration of VLAN ID per virtual stream. (ID: 11415083)

## 3.4.6 VLAN ID re-configuration in eFramer

### 3.4.6.0-1 VLAN ID re-configuration in eFramer L1SW shall configure VLAN ID per eFramer stream based on the contents of API2_L1SW_CONFIGURE_TRANSPORT_REQ_MSG. L1SW shall support re-configuration of VLAN ID in runtime during cell deletion and addition procedure. L1SW shall configure the default PCP values (i.e. PCP value = 7) for Fast C-Plane and U-Plane streams. (ID: 11415087)
