### 7.2.4.1 Guard Period Length Configuration

#### 7.2.4.1.0-1 Guard Period Length Configuration L1 shall support configuration of guard period length values to accommodate extended cell coverage scenarios up to 11 km. The configuration shall support 6-symbol and 10-symbol guard period (GP) values within the 4-4 frame structure, ensuring alignment with the propagation delays of long-range UEs in FR2. Rationale: Adjustable guard period lengths are essential to maintain DL-UL frame alignment in extended-range cells, avoiding timing conflicts and ensuring reliable TDD operation. (ID: 12733457)

### 7.2.7 L1-L2/L3 Interface Mapping for Extended Frame Configuration

#### 7.2.7.0-1 L1-L2/L3 Interface Mapping for Extended Frame Configuration L1 shall process and apply configuration parameters received via L2/L3 interface related to the 4-4 frame structure for extended cell coverage. The interface shall include timing adjustments, guard period configurations, and frame alignment controls ensuring synchronization integrity across cell extended-range UEs. Rationale: Required to ensure L1 behavior aligns with higher-layer configuration for 4-4 frame operation in FR2 extended cell, enabling seamless integration with L2/L3 control mechanisms and maintaining accurate frame structure handling up to 11km.
