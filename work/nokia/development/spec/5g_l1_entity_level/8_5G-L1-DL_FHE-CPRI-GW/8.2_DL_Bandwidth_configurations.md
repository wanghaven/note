# 8.2 DL Bandwidth configurations

**8.2.0-1**  (ID: `13255795`)

DL Bandwidth configurations
5G-L1-DL (FHE-CPRI-GW) shall support the following DL bandwidth configuration for FR1 FDD:

* DL bandwidth is provided by bandwidth in API\_BBB\_CREATE\_EAXC\_AXC\_ROUTING\_REQ\_MSG  (with direction is "EBbbEaxcDirection\_Downlink")
* Sample Rate is provided by iqSampleCount in API\_BBB\_CREATE\_EAXC\_AXC\_ROUTING\_REQ\_MSG  (with direction is "EBbbEaxcDirection\_Downlink")
