**Title:** L2-RT Slot Blanking Configuration for RF Cooling
**Participants:**
\* 5G-L2-PS
\* 5G-CP-RT
**Sequence of Events:**
1. **(10) PsCell\_SlotBlankingReq:** A request message is sent from 5G-L2-PS to 5G-CP-RT. This message contains the following parameters:
\* `rfCoolingMode`
\* `rfUnitSharedWithAnotherCell`
\* `tgtMutingRatio`
2. **(20) L2-PS store and initialize slot blanking info.:** This is an internal action or note associated with the 5G-L2-PS participant. It states that based on the `rfCoolingMode` value, `tgtMutingRatio`, and `achievedMutingRatio` will be set.
3. **(30) PsCell\_SlotBlankingResp:** A response message is sent from 5G-CP-RT back to 5G-L2-PS.
