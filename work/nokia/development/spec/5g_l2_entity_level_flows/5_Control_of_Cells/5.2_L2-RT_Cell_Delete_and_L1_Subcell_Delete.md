2. **Positioning cell context cleanup**: 5G-CP-RT performs a cleanup of the positioning cell context.
3. **PsPos\_PosCellDeleteResp**: 5G-CP-RT sends a response back to 5G-L2-PS, indicating the successful deletion of the positioning cell.
4. **UICell\_DeleteReq**: 5G-L2-PS sends a request to 5G-L1-UL to delete a UI cell.
5. **UICell\_DeleteResp**: 5G-L1-UL sends a response back to 5G-L2-PS, confirming the deletion of the UI cell.[/image\_desc][image\_path]https://storage.googleapis.com/dng\_files/images/HTML\_m1dng\_5G\_L2\_Entity\_Level\_Flows\_file\_222.001.png[/image\_path][/image-0]
