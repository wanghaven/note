## 15.2.2 Description

### 15.2.2.0-1 Description (1) DlData_PdcchSendReq (ID: 11207907)

**15.2.2.0-1.0-1**  (ID: `11207915`)

Description
Conditions to trigger the CG deactivation are fullfilled (refer to 5G\_L2\_11021\_replaced([11175928](https://dn-prod.ext.net.nokia.com/rm/resources/BI_26wGFPvTEe-AqvopbP1qhQ))).

A PDCCH candidate to send the DCI 0\_0 for CG deactivation is available.

The DlData\_PdcchSendReq message is sent to 5G-L1-DL with the following setting:

* rnti = CS-RNTI
* dciScramblingRnti = C-RNTI
* dciPayload = DCI 0\_0 CG deactivation with CG deactivation format (refer to 5G\_UP\_ALG\_14812\_replaced([11172948](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kuPHzfvSEe-AqvopbP1qhQ)))

[CB008124-C]

Counter UL\_CG\_DEACT\_ATT M55308C00022 shall be incremented.

[End CB008124-C]

### 15.2.2.0-5 Description (5) DlData_PdcchSendReq (ID: 11207979)

**15.2.2.0-5.0-1**  (ID: `11207988`)

Description
A UL grant is required to allow the UE to send its MAC CE Configured Grant Confirmation related to the CG deactivation confirmation.

A PDCCH candidate to send the UL grant is available.

The DlData\_PdcchSendReq message is sent to 5G-L1-DL with the following setting:

* rnti = C-RNTI
* dciScramblingRnti = C-RNTI
* dciPayload = UL grant

### 15.2.2.0-7 Description (7) UlData_PuschReceiveReq (ID: 11208017)

**15.2.2.0-7.0-1**  (ID: `11208027`)

Description
The UlData\_PuschReceiveReq is sent to 5G-L1-UL to receive the PUSCH possibly containing the MAC CE Configured Grant Confirmation related to the CG deactivation confirmation from UE, with the following setting:

* rnti = CS-RNTI or C-RNTI depending on condition (3) or (4) respectively
* cRnti = C-RNTI
* PRB resources (i.e. startPrb and numOfPrb) are set to

- the CG PRB resources if the Uldata\_PuschReceiveReq is sent k2 slots before a non-blocked CG slot (i.e. re-use of CG PRB resources). If CG slot is blocked, CG RBs are not available anymore.

- the new PRB resources allocated if the Uldata\_PuschReceiveReq is sent k2 slots before a non-CG slot

### 15.2.2.0-11 Description (11) UlData_PuschReceiveRespLo (ID: 11208087)

**15.2.2.0-11.0-1**  (ID: `11208095`)

Description
5G-L1-UL returns to 5G-L2-LO the UlData\_PuschReceiveRespLo message with the following setting:

* rnti = C-RNTI
* data = TB received from UE and most likely containing the MAC CE Configured Grant Confirmation related to the CG deactivation confirmation

---

### 15.2.2.0-12 Description (12) LoCtrl_UlMacPduReceiveInd (ID: 11208100)

**15.2.2.0-12.0-1**  (ID: `11208110`)

Description
5G-L2-LO detects, in the TB received from UE, the MAC CE Configured Grant confirmation related to the CG deactivation confirmation.

5G-L2-LO sends the LoCtrl\_UlMacPduReceiveInd with isConfiguredGrantContCe = True to inform 5G-L2-PS that CG deactivation is completed.

From this moment, 5G-L2-PS shall consider that the UE is returned in CG configured state.

[CB008124-C]

Counter UL\_CG\_DEACT\_ATT\_SUCC M55308C00023 shall be incremented.

[End CB008124-C]
