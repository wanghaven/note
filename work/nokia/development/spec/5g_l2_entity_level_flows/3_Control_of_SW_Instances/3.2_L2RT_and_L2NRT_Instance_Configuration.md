Configuration sequence:
[image-0][image\_desc]The image is a sequence diagram illustrating the L2RT and L2NRT Instance Configuration process. The diagram shows interactions between different network functions, including 5G-L1-UL, 5G-L1-DL, 5G-L2-PS, 5G-L2-LO, 5G-L2-HI, and 5G-CP.
The configuration process is initiated by OAM, which triggers CP. The diagram then enters a loop that iterates for each L2RT Pool in an L2RT instance. Within the loop, there are conditional messages related to whether the L2RT pool is dedicated for positioning.
The core interactions involve address distribution requests and responses:
- `LoCnfg_AddressDistributionReq` and `LoCnfg_AddressDistributionResp` are exchanged for LoUser and LoCtrl addresses.
- `PsCnfg_AddressDistributionReq` and `PsCnfg_AddressDistributionResp` are exchanged for PsUser, PsSgnl, PsCell, and PsPos addresses.
Following this, there are parallel (par) sequences of messages:
- The first parallel sequence involves `HiCnfgCu_AddressDistributionReq` and `HiCnfgCu_AddressDistributionResp`. This exchange provides addresses of HiUser and Hisgnl, bearer group information, and addresses for GTP-u packet exchanges on the F1-U interface.
- The second parallel sequence involves `HiCnfgDu_AddressDistributionReq` and `HiCnfgDu_AddressDistributionResp`. This exchange provides bearer group information, addresses for uplink PDU exchanges, flow control exchanges, F1-U EQ addresses for downlink GTP-U packet exchange, and HiMeasDu Address exchanges.
Finally, there are two more parallel sequences for configuration parameters:
- The first involves `HiCnfgCu_ConfigParamReq` and `HiCnfgCu_ConfigParamResp`, where configuration parameters are provided to L2HI CU.
- The second involves `HiCnfgDu_ConfigParamReq` and `HiCnfgDu_ConfigParamResp`, where configuration parameters are provided to L2Hi DU.
The diagram clearly outlines the flow of information and the sequence of operations for configuring L2RT and L2NRT instances.[/image\_desc][image\_path]https://storage.googleapis.com/dng\_files/images/HTML\_m1dng\_5G\_L2\_Entity\_Level\_Flows\_file\_37.001.png[/image\_path][/image-0]
