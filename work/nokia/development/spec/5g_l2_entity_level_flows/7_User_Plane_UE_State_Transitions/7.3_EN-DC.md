###### 7.3.4.4.2.0-25 Description: (25) DL GTP-U PDU (UL PDCP Status Report) (ID: 13337031)

**7.3.4.4.2.0-25.0-1**  (ID: `13337032`)

Description:
PDCP Control PDU: PDCP Status Report for UL data is indicating (per bearer) to UE which PDCP PDUs are missing or successfully received.

Note:

In case dlDataSplitMode parameter (received in the HiUserCu\_BearerSetupReq--[SplitBearerParams](https://dn-prod.ext.net.nokia.com/rm/resources/TX_CyLl4Pu6Ee-AqvopbP1qhQ) message) is equal to overF1UXxU, then 5G-L2-HI shall select the transmission link with the following priority:

1. The link over which the first UL GTP-U PDU was received
2. in case of PDCP data recovery (pdcpKeyChangeOngoing = False) UL GTP-U PDU from F1-U (step 15)
3. in case of PDCP re-establishment (pdcpKeyChangeOngoing = True) UL GTP-U PDU from F1-U or X2-U tunnel (step 15 or 16)
4. The primary link (target F1-U tunnel).
5. If the primary link is suspended, the secondary link (target X2-U tunnel).
