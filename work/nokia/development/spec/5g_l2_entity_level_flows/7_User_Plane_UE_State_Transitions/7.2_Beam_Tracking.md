1.5G-L2-PS sends UlData\_PrachReceiveReq message with PRACH time and frequency positions.
2. The 5G-L1-UL sends the list of detected random access preambles to 5G-L2-PS in UlData\_PrachReceiveInd message.
3. 5G-L2-PS retrieves SS Block Index from PRACH time occasion and PRACH frequency occasion and deduce beam ID from SS block index, see 5G\_L2\_3577\_replaced([11153498](https://dn-prod.ext.net.nokia.com/rm/resources/BI_24us1fvTEe-AqvopbP1qhQ)).
4. 5G-L2-PS sends UlData\_PuschReceiveReq(5G\_L1\_IF\_183\_replaced([11513708](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FRyW7_vxEe-AqvopbP1qhQ)))/UlData\_PucchReceiveReq(5G\_L1\_IF\_212\_replaced([11513942](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FR0MFfvxEe-AqvopbP1qhQ)) to 5G-L1-UL and receives  UlData\_PuschReceiveRespPs(5G\_L1\_IF\_200\_replaced([11513831](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FRzlAPvxEe-AqvopbP1qhQ)))/UlData\_PucchReceiveRespPs (5G\_L1\_IF\_221\_replaced([11514182](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FR2BQvvxEe-AqvopbP1qhQ))) from 5G-L1-UL. BeamformingMeasurementMetric(5G\_UP\_ALG\_1992\_replaced([11112073](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpS9Y_vSEe-AqvopbP1qhQ))) shall be calculated per measured beam based on CRI-RSRP/SSBRI-RSRP and corresponding CRI/SSBRI.
5. Power value for each beam shall be filtered according to 5G\_UP\_ALG\_1999\_replaced([11112098](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpS9avvSEe-AqvopbP1qhQ)).
6. Target beam for the next beam change is selected according to 5G\_UP\_ALG\_2006\_replaced([11112105](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpS9bPvSEe-AqvopbP1qhQ)) or in 5G\_UP\_ALG\_1992\_replaced([11112073](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpS9Y_vSEe-AqvopbP1qhQ)).
7. TCI State Indication (5G\_UP\_ALG\_2103\_replaced([11112673](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpYc9PvSEe-AqvopbP1qhQ))) for UE-specific PDCCH MAC CE(5G\_L2\_1974\_replaced([11215724](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2-6NUvvTEe-AqvopbP1qhQ)))  shall be sent if a new target SSB beam is selected. The updated beam ID shall be applied to physical channels when MAC CE PDCCH TCI state indication is HARQ ACKed or finally HARQ NACKed plus a fixed delay, see 5G\_UP\_ALG\_2032\_replaced([11112298](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kpUymPvSEe-AqvopbP1qhQ)).
8.   UL beam probing is exectued before new beam is applied as further serving beam for UL, see 5G\_L2\_8180\_replaced([11153774](https://dn-prod.ext.net.nokia.com/rm/resources/BI_24wiBvvTEe-AqvopbP1qhQ))/8176.
      9.   UL beam sweeping is triggered if it's considered as beam lost, see (5G\_L2\_7995\_replaced([11153786](https://dn-prod.ext.net.nokia.com/rm/resources/BI_24wiCfvTEe-AqvopbP1qhQ))).
