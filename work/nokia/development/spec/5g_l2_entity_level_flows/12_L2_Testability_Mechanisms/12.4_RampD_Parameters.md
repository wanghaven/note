# 12.4 R&amp;D Parameters

**12.4.0-1**  (ID: `11206449`)

R&D Parameters
R&D Parameters are internal parameters not visible by the client.

R&D Parameters are configured and modified via configuration files.

If a local configuration file is present, it takes precedence over the global configuration file, and this local file can be modified in 5G-L2-HI, 5G-L2-LO and 5G-L2-PS by AaConfig API (platform CC&S services).

R&D parameters are intended to be lightweight so that they can be used extensively from applications and CC&S SW.

R&D parameter is discriminated by a Tag, managed by AaConfigTag.

Name based attribute tag key:

* Maximum length of attribute tag name is 80 bytes
* Tag key discrimination using namespace concept

Variable sized attribute tag data

* Maximum size of attribute tag data is 256 bytes
* Supported internal data types: u16, u32, string, any
