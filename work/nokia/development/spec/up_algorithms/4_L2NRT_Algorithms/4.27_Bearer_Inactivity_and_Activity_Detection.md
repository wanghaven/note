# 4.27 Bearer Inactivity and Activity Detection

## 4.27.0-2 Bearer Inactivity and Activity Detection Inactivity Detection (ID: 11228392)

**4.27.0-2.0-3**  (ID: `11228414`)

Bearer Inactivity and Activity Detection On request, 5G-L2-HI shall perform the bearer inactivity detection.

5G-L2-HI shall perform the bearer inactivity detection only when bearerInactivityTimer is present.

bearerInactivityTimer should be present for 5QI-1 DRB to support bearer inactivity detection.

5G-L2-HI shall detect inactivity for a bearer, if there is no data in the buffer, and no data received for the bearer both in DL and UL for time period of “Inactivity Time”, the time duration for which the inactivity is monitored. 

“Inactivity Time”: The time duration for which the inactivity is monitored.

The value of “Inactivity Time” is defined by bearerInactivityTimer or rdBearerMinimumInactivityTimer (Note 2).

For EN-DC/NSA, bearerInactivityTimer is defined by NRBTS.nsaInactivityTimer

For SA, bearerInactivityTimer is defined by NRBTS.inactivityTimerSA

Besides common bearerInactivityTimer (NIDD parameter) for the others, NRBTS.drbInactivityTimerSA5qi1(NIDD parameter) is for 5QI-1 DRB and NRBTS.drbInactivityTimerSA5qi5 (NIDD parameter) is for 5QI-5 DRB.

[CB008118-A-UPD010]For Mission Critical bearers 5QI-65, 5QI-66, 5QI-67, 5QI-69 and 5QI-70, NIDD parameter NRBTS.NRDRB.drbInactivityTimerSA shall be referred. [End CB008118-A-UPD010]

On the detection of inactivity for a bearer 5G-L2-HI shall send a message to control plane.

Note:

1\. Received but immediately discarded packets are also considered as received.

2\. In case 5G-L2-HI receives a “Inactivity Time” of value zero, it applies a default value instead.

## 4.27.0-3 Bearer Inactivity and Activity Detection Activity Detection (ID: 11228422)

**4.27.0-3.0-1**  (ID: `11228429`)

Bearer Inactivity and Activity Detection 5G-L2-HI shall monitor the bearer for any further activity after sending the inactivity message.

On receipt of first DL PDCP SDU or UL PDCP PDU for the bearer for which inactivity message was sent, 5G-L2-HI shall inform CPlane about the activity of the bearer and start inactivity monitoring as per 5G_UP_ALG_6204_replaced([11228392](https://dn-prod.ext.net.nokia.com/rm/resources/BI_kyhx8PvSEe-AqvopbP1qhQ)).
