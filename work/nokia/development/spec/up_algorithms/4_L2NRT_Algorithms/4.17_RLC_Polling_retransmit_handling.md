# 4.17 RLC Polling retransmit handling

## 4.17.0-2 RLC Polling retransmit handling Starting of t-PollRetransmit timer and POLL SN setting for DRB mapped to RLC AM (ID: 11225499)

**4.17.0-2.0-2**  (ID: `11225511`)

RLC Polling retransmit handling 5G-L2-HI shall perform the below action,

5G-L2-HI shall perform the below action only when between the previous and current [Before CNI-88099-F]LoData_UeDataDeliveryStatusInd[CNI-88099-F] 5G-L2-LO DDDS indication [End CNI-88099-F]

 RLC STATUS PDU has not been received OR

 RLC PDU with RLC SN = pollSnSubmittedToMac has not been positively or negatively acknowledged in the received RLC STATUS PDU OR

 RLC PDU with RLC SN = pollSnSubmittedToMac has been negatively acknowledged in the received RLC STATUS PDU and RLC retransmission has been sent to 5G-L2-LO.

Incase t-PollRetransmit is not running

  * Set POLL_SN = pollSnSubmittedToMac
  * Start the t-PollRetransmit timer.



Incase t-PollRetransmit is running AND POLL_SN <= pollSnSubmittedToMac

  * Set POLL_SN = pollSnSubmittedToMac
  * Restart the t-PollRetransmit timer.



pollSnSubmittedToMac is received in [Before CNI-88099-F]LoData_UeDataDeliveryStatusInd[CNI-88099-F] 5G-L2-LO DDDS indication [End CNI-88099-F].

## 4.17.0-3 RLC Polling retransmit handling Starting of t-PollRetransmit timer and POLL SN setting for SRB mapped to RLC AM (ID: 11225518)

**4.17.0-3.0-3**  (ID: `12801097`)

RLC Polling retransmit handling Upon receiving a 5G-L2-LO SRB data delivery status indication message from 5G-L2-LO, the 5G-L2-HI shall perform the below action only when between the previous and current 5G-L2-LO SRB data delivery status indication 

· RLC STATUS PDU has not been received OR

· RLC PDU with RLC SN = pollSnSubmittedToMac has not been positively or negatively acknowledged in the received RLC STATUS PDU OR

· RLC PDU with RLC SN = pollSnSubmittedToMac has been negatively acknowledged in the received RLC STATUS PDU and RLC retransmission has been sent to 5G-L2-LO.

Incase t-PollRetransmit is not running

  * Set POLL_SN = pollSnSubmittedToMac
  * Start the t-PollRetransmit timer.



Incase t-PollRetransmit is running AND POLL_SN <= pollSnSubmittedToMac

  * Set POLL_SN = pollSnSubmittedToMac
  * Restart the t-PollRetransmit timer.



[04956604]

If NRBTS/actEnhRrcReconfUlPrbRandom = True,

For SRB1, when the t-PollRetransmit timer expires, all the RLC PDUs with SN less than that of the RLC_SN_with_PollBit_set, that were not successfully acknowledged shall be retransmitted.

RLC_SN_with_PollBit_set = the RLC SN of the latest RLC PDU sent out of gNB-DU (5G-L2-HI) with poll bit set.

[End 04956604]

pollSnSubmittedToMac is received in 5G-L2-LO SRB data delivery status indication.
