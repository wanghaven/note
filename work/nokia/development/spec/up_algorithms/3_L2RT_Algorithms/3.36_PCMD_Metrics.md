  * The downlink RLC PDU (MAC SDU) burst volume: gNB-DU shall add this metric to the trace record entry specified in [12455939](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FLJO5wpAEfCT35inaBGo7g).

[CB013789-SR-E]

For FR1, 5G-L2-HI reports these PCMD metrics based on the "Burst Info Report" [12481592](https://dn-prod.ext.net.nokia.com/rm/resources/TX_lYhdEA9pEfCvOvZoOJBSnQ) from 5G-L2-LO. If the PCMD Framework content is changed online, 5G-L2-LO will start collecting and sending the "Burst Info Report" from the beginning of the next PCMD report period.   
For FR2, those PCMD metrics are reported by 5G-L2-LO.

[End CB013789-SR-E]

Burst volume/burst time mentioned above is based on the merged burst across all the component carriers.
