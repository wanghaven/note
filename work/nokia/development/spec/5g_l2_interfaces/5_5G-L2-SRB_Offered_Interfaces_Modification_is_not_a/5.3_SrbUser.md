#### 5.3.2.1.0-3 Bearer Setup Message Validation (ID: 10993142)

**5.3.2.1.0-3.0-1**  (ID: `10993148`)

Bearer Setup
Upon reception of the SrbUser\_BearerSetupReq message, 5G-L2-SRB shall check the validity of the request message.

5G-L2-SRB shall reply with SrbUser\_BearerSetupResp message and set the status IE = Ok if the request message is valid and the bearer was setup successfully.

For exception handling, see 5G\_L2\_IF\_1335\_replaced([10993154](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2W8Xzvu6Ee-AqvopbP1qhQ)).

#### 5.3.2.1.0-4 Bearer Setup Exceptions (ID: 10993154)

**5.3.2.1.0-4.0-1**  (ID: `10993160`)

Bearer Setup
[5GC002184-I]

Expected Exceptions:

Generic exception handling rules can be found in the chapter 5G\_UP\_1795\_replaced([11374912](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueKrmfvkEe-AqvopbP1qhQ)).

Exception handling rules specifically for object setup can be found in the requirements 5G\_UP\_1805\_replaced([11375008](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueLSpvvkEe-AqvopbP1qhQ)) and 5G\_UP\_1806\_replaced([11375035](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueLSqfvkEe-AqvopbP1qhQ)).

[End 5GC002184-I]

[Before 5GC002184-I]

Expected Exceptions:

5G-L2-SRB shall return NOK in case of the following expected exceptions

1. bearerIndex is already allocated

2. bearerIndex is out of range

3. srbIdentity is out of range

4. Security Parameters are not defined for SRB3

5. Unable to allocate a context for the bearer

5G-L2-SRB shall return OK in case a context for the bearer is setup succesfully.

Unexpected Exceptions:

In case of unexpected exception, 5G-L2-SRB may not send a response message.

[End 5GC002184-I]

[5GC002184-I]

5G-L2-SRB shall reply with SrbUser\_BearerSetupResp message, set the status IE = NotOk, and set an appropriate cause value if:

* bearerIndex IE with same index exists already or;
* bearerIndex IE out of range or;
* srbIdentity IE out of range or;
* Security parameters (structure) are not defined for SRB3 or;
* unable to allocate a context for the bearer or;
* the request message is not valid or;
* the resources were not allocated successfully.

For available cause values, see 5G\_UP\_1846\_replaced([11375790](https://dn-prod.ext.net.nokia.com/rm/resources/BI_uePkGPvkEe-AqvopbP1qhQ)).

Unexpected Exceptions:

In case of an unexpected exception, 5G-L2-SRB shall not send response message.

Error is logged for both expected and unexpected exception.

[End 5GC002184-I]
