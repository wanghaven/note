## 2.8.2 Procedure

### 2.8.2.1 UL Radio Link Status Indication

#### 2.8.2.1.0-1 UL Radio Link Status Indication Overview (ID: 10979085)

**2.8.2.1.0-1.0-1**  (ID: `13074316`)

UL Radio Link Status Indication
Messages:

PsCtrl\_UlRadioLinkStatusInd

Purpose:

To indicate Radio link outage or Resume to RLC.

Precondition:

User context exists and radio link status detection is started in 5G-L2-PS.

Addressing:

The indication is sent to the PsCtrl\_RlStatusClient address(IE psCtrlRlStatusClientAddress in drbAddReqPs, refer to 5G\_L2\_IF\_397\_replaced([10974876](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2U-o4vu6Ee-AqvopbP1qhQ))).

Relevant UP MSCs:

5G\_L1L2\_FLOW\_563\_replaced([11200643](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BgNwQvvREe-AqvopbP1qhQ)): UL user Data Split Mode, radio link status change sequence based on UL radio quality

5G\_L1L2\_FLOW\_565\_replaced([11200664](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BgNwTvvREe-AqvopbP1qhQ)): UL user Data Split Mode change based on the UL radio link quality

#### 2.8.2.1.0-2 UL Radio Link Status Indication Description (ID: 10979090)

**2.8.2.1.0-2.0-1**  (ID: `13074320`)

UL Radio Link Status Indication
5G-L2-PS shall send an indication of UL radio link status to 5G-L2-HI (DU). See 5G\_L1L2\_FLOW\_563\_replaced([11200643](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BgNwQvvREe-AqvopbP1qhQ)).

Upon reception of the request, 5G-L2-HI (DU) shall send UL RLO/ UL RESUME to PDCP entity of 5G-L2-HI via DDDS as defined in 5G\_UP\_ALG\_5813

---

#### 2.8.2.1.0-3 UL Radio Link Status Indication Message validation (ID: 10979095)

**2.8.2.1.0-3.0-1**  (ID: `13074329`)

UL Radio Link Status Indication
Upon reception of the PsCtrl\_UlRadioLinkStatusInd message, 5G-L2-HI shall check the validity of the indication message.

For exception handling, see 5G\_L2\_IF\_635\_replaced([10979100](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2VYRjvu6Ee-AqvopbP1qhQ)).

#### 2.8.2.1.0-4 UL Radio Link Status Indication Exceptions (ID: 10979100)

**2.8.2.1.0-4.0-1**  (ID: `13074341`)

UL Radio Link Status Indication
Expected Exceptions:

[5GC002184-E]

Generic exception handling rules can be found in the chapter 5G\_UP\_1795\_replaced([11374912](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueKrmfvkEe-AqvopbP1qhQ)).

5G-L2-HI shall discard the message if:

* bearerIndex IE is not known or;
* bearerGroup IE is not known or;
* the request message is not valid.

[End 5GC002184-E]

Unexpected Exceptions:

[5GC002184-E]

In case of an unexpected exception, 5G-L2-HI shall discard the message.

Error is logged for both expected and unexpected exception.

[End 5GC002184-E]
