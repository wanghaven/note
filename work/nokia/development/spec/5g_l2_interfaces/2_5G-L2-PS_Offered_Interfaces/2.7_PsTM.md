Beam Pattern Change
Messages:

PsTM\_BeamPatternChangeReq/Resp

Purpose:

This procedure is to change the beam pattern for an ongoing test for FR1 TDD and FR2. 5G-L2-PS shall apply the new beam pattern to the test from next occurence of transmission or receiving after it receives PsTM\_BeamPatternChangeReq. 5G-L2-PS shall autonomously apply the same beam pattern for given direction(UL or DL) for all the avaliable cells which are listed in nrCellIdentityList[].

Precondition:

Test is already started for the specific test type on the specific cell.

Addressing:

The request is sent to the PsTM address.

The response is sent to the address where the request was sent from.
