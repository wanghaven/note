#### 4.7.2.1.0-3 Srb Send Exceptions (ID: 10991074)

**4.7.2.1.0-3.0-1**  (ID: `10991081`)

Srb Send
Expected:

Generic exception handling rules can be found in the chapter 5G\_UP\_1795\_replaced([11374912](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ueKrmfvkEe-AqvopbP1qhQ)).

5G-L2-HI shall discard the message if:

* bearerIndex IE is not known (bearer does not exist) or;
* bearerGroup IE is not known or;
* logicalChannelId IE is not known (bearer does not exist) or;
* the request message is not valid.

Note: 5G-L2-HI shall not discard the HiSgnlDu\_SrbSendReq for validation errors related the parameter rrcMessageIdentifier, but only log an error.

Rationale: This IE is transparent for 5G-L2-HI and can be set to unused value, see 5G\_L3\_5554\_replaced([11068112](https://dn-prod.ext.net.nokia.com/rm/resources/BI_BDILj__KEe-D3seUwaI0tg)).

Unexpected:

In case of an unexpected exception, 5G-L2-HI shall discard the message.

Note:

HiSgnlDu\_SrbSendResp message is sent only in those cases where response is requested by the client.

See 5G\_L2\_IF\_1242\_replaced([10991088](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2Wvjd_u6Ee-AqvopbP1qhQ)) and 5G\_L2\_IF\_1243\_replaced([10991122](https://dn-prod.ext.net.nokia.com/rm/resources/BI_2WvjfPu6Ee-AqvopbP1qhQ)) for details. Response message shall not be sent if the message is discarded based on this requirement.

[PR657986]

5G-L2-HI shall reply with HiSgnlDu\_SrbSendResp message also if HiUserDu\_BearerErrorInd has been originated and respFlag is set to true in HiSgnlDu\_SrbSendReq.

[End PR657986]
