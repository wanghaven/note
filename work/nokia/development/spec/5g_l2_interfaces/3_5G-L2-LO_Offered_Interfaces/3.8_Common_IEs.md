# 3.8 Common IEs

## 3.8.0-2 Common IEs SfnTimeStamp (ID: 12774333)

**3.8.0-2.0-1**  (ID: `12774335`)

Common IEs
SfnTimeStamp u64 with following bitfields:

|  |  |  |  |
| --- | --- | --- | --- |
| Bits | 63     ...     48 | 47     ...     32 | 31     ...     0 |
| Size | 16 bits | 16 bits | 32 bits |
| Parameter | hyperFrameNr | frameNr | btusInFrameNr |
| Range | 0...1023 | 0...1023 | 0...19660799 |
| Description | System Hyper Frame Number. | System Frame Number. | Basic time unit number since frame beginning. Unit is 1/19660800 of frame duration. |

Note:

1. btusInFrameNr = slotNr \* slotDurationUs / btuDurationUs + btuNr

btuDurationUs = frameDurationUs(=10000) / btusInFrame(=19660800)
