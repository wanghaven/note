**[5GC000579-B][PR449788][PR449748] 5G-L1-DL p pointer negotiation**

OAM shall configure CPRI links with cpriPointerP to 0 corresponding to CPRI passive link (no C&M capability).

OAM shall configure CPRI links with a minimal cpriPointerP to 40 corresponding to the minimal p value to be negotiated with RU (with C&M capability).

for active C&M cpri links

5G-L1-DL shall:

* Initalize the CPRI link p pointer (Z.194.0) to the cpriPointerP provided by OAM from API\_L1SW\_CPRI\_CONFIGURE\_LINKS\_REQ\_MSG
* Once the cpri link state is entering at least State D:

read the p pointer reported back from slave port (CPRI rx side Z.194.0 value)

if p pointer reported by slave port == 0

then 5G-L1-DL will set its CPRI master port p pointer to 0 and the link will be initialized as a passive link (no C&M)

if p pointer reported by slave port > p pointer of the master port

then 5G-L1-DL will set its CPRI master port p pointer to the value reported by slave port
