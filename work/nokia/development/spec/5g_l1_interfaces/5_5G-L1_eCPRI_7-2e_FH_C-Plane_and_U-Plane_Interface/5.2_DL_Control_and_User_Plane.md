## 5.2.5 Slot Type Configuration

### 5.2.5.1 Procedure

**5.2.5.1.0-1**  (ID: `11516946`)

Procedure
Messages:

DlDataFH\_SlotTypeReq

Purpose:

Provides with a parameter which enable to apply a power offset or to mute a symbol inside a slot -useful for DL signal only- for test purpose or graceful cell shut down.

Provides too with the slot type field which, for each NR-slot, describes the signal expected for each symbol.

Provides power reduction only when it has non-zero value for at least one symbol in the slot.

Precondition:

Subcell exists and L1 downlink addresses have been negotiated for it [5G\_L1\_IF\_37\_replaced([11512324](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FRol5fvxEe-AqvopbP1qhQ))]

Addressing:

No addressing needed between L1BB and L1RU or viceversa. Not relevant for 7.2e FH split.

There is no response related to the request.
