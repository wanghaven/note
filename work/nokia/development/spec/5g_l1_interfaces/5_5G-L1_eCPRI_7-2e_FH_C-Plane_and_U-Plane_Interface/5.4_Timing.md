## 5.4.2 L1 uplink response messages transmission timing window

**5.4.2.0-2**  (ID: `11517658`)

L1 uplink response messages transmission timing window
The timing reference for C-plane messages (slot-wise) is air time for the start of the UL slot.

The timing reference for U-plane messages (symbol-wise) is air time for the start of the respective UL symbol slot.

The values for **C+Ta4\_min\_cp and** D+Ta4\_max\_cp, C+Ta4\_min\_up and **D+Ta4\_max\_up,** are in in the table below, see columns *Earliest sending time to L2 and* Latest sending time to L2, respectively.***\****\*\*\*

|  |  |  |
| --- | --- | --- |
| **L1-L2 message** | **Earliest sending time to L2(µs).** | **Latest sending time to L2 (µs).** |
| UlData\_PuschReceiveRespPs | Ta4\_min\_cp | 500+Ta4\_max\_cp |
| UlData\_PuschReceiveRespLo | Ta4\_min\_up+tUL(13) | 1000+Ta4\_max\_up+tUL(13) |
| UlData\_PuschReceiveRespHarqU | Ta4\_min\_up+tUL(13) | 500+Ta4\_max\_up+tUL(13) |
| UlData\_PuschReceiveRespHarqD | Ta4\_min\_up+tUL(13) | 500+Ta4\_max\_up+tUL(13) |
|  |  |  |
| UlData\_SrsReceiveRespPs | Ta4\_min\_cp | 500+Ta4\_max\_cp |
| UlData\_SrsReceiveRespBmPs | Ta4\_min\_cp | 500+Ta4\_max\_cp |

A)Ta4\_min\_cp and Ta4\_max\_cp are used for messages: UlData\_PuschReceiveRespPs, UlData\_SrsReceiveRespPs, UlData\_SrsReceiveRespBmPs. Note that Pusch and Srs have different Ta4\_min\_cp and Ta4\_max\_cp.

B)tul(13)+Ta4\_min\_up and tul(13)+Ta4\_max\_up, i.e., the relative window for the last symbol in the slot, are used for messages to L2LO: UlData\_PuschReceiveRespLo, UlData\_PuschReceiveRespHarqU, UlData\_PuschReceiveRespHarqD.

The values for **Ta3\_min\_cp,** Ta3\_max\_cp, **Ta4\_min\_cp,** Ta4\_max\_cp, Ta3\_min\_up, Ta3\_max\_up, Ta4\_min\_up, Ta4\_max\_up,  and **T34\_max are specified in document *eCPRI LL Split U-Plane Application Layer Specification. Latest version can be found at: [12739988](https://dn-prod.ext.net.nokia.com/rm/resources/BI_SQ7TEHnCEfCiUemk-K-C4g)******\****\*

The value of NTAoffset is defined in 5G\_L1\_IF\_282\_replaced([11514759](https://dn-prod.ext.net.nokia.com/rm/resources/BI_FR5roPvxEe-AqvopbP1qhQ)).
