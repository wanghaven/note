[CB014419-SR-EB] There is one L2-RT instance in an ABIR for L3 call on Host. [End CB014419-SR-EB]

There can be up to two L2-RT instances in other hardwares. See 5G_UP_2369_replaced([11386513](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ufC1WPvkEe-AqvopbP1qhQ)).

Inside one L2-RT instance (cf. 5G_UP_2433_replaced([11388475](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ufKKHvvkEe-AqvopbP1qhQ))):

  * In TDD FR2, Each NRCELLGRP is handling one or more cells.
  * In TDD FR1, there is always one cell per NRCELLGRP. That means that deployment referenced as one sector two cells in SISO configuration block is deployed as 2 NRCELLGRP each one with one cell.
  * In FDD, there is always one cell per NRCELLGRP.



Refer to 5G_UP_2545_replaced([11390823](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ufVwSPvkEe-AqvopbP1qhQ)) for Max Cells number per L2-RT instance for each CellTechnology (TDD, FDD) with different NrCellType on different Board Type.

**Note:**

5G_UP_2423_replaced([11388385](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ufKKEfvkEe-AqvopbP1qhQ)) defines the DeploymentId which should be used by 5G-L2-PS in the half ABIO, ASOE and ASOF on TDD FR2.
