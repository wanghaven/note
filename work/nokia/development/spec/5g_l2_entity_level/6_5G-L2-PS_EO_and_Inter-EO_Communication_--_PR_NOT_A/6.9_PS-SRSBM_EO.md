### 6.9.1.1 Asynchronous Event Processing functionalities

#### 6.9.1.1.0-1 Asynchronous Event Processing functionalities [CNI-80481]SRSBM EO handles following messages / events ∙    PsInternal messages / events forwarded by PS Signaling EO, which is usually a direct copy of external messages from other SCs, e.g. PsInternal_CellSetupReq, PsInternal_UserSetupReq, PsInternal_CellReconfigurationReq, PsInternal_CellStopSchedulingReq,   PsInternal_UserModifyReq, PsInteral_UserDeleteInd, etc. ∙    Messages / Events forwared by PS-UL EO, which is a direct forward of external message from L1-UL, e.g. UlData_SrsReceiveRespBmPs. [END CNI-80481] (ID: 11246664)

### 6.9.1.2 SlotSynchroInd Event handling

#### 6.9.1.2.0-1 SlotSynchroInd Event handling [CNI-80481] Following functionalities are invovled: ∙   Timer wheel handling to check if rrmDlBeamSelectTimer and rrmCoMaAgedTimer expires. If rrmCoMaAgedTimer expires, execute 5G_L2_6302_replaced( 11154626 ). ∙   Beam selection to calculate best and second-best beams if rrmDlBeamSelectTimer expires, see 5G_L2_6564_replaced( 11154888 ). [END CNI-80481] (ID: 11246678)
