## 6.7.2 FD Scheduler

### 6.7.2.0-1 FD Scheduler Send pointer to DlData_PdcchSendReq to PS-DL EO (ID: 11245990)

**6.7.2.0-1.0-1**  (ID: `11245994`)

FD Scheduler As described in 5G_L2_278 and 5G_UP_ALG_4466_replaced([11178897](https://dn-prod.ext.net.nokia.com/rm/resources/BI_ku2y0fvSEe-AqvopbP1qhQ)), calculation of counter DAI is done after FD scheduling. Because of FD parallelization in FR2, the counter DAI calculation is done in DL EO, meaning after the creation of DlData_PdcchSendReq done in PS-FD EO

FD EO will send pointers to the created DlData_PdcchSendReq to PS-DL EO by filling field pdcchEventPtr in the message PsInternal_FdScheduleResp defined in 5G_L2_5787.
