# 5.2 L3 Message Processing

## 5.2.0-6 L3 Message Processing 5G-L2-SRB shall receive SrbSgnl_ShortMacVerificationReq message 1. Upon reception SrbSgnl_ShortMacVerificationReq message, 5G-L2-SRB shall invoke integrity verification as per 5G_L2_2800_replaced( 11240989 ) on the Message payload using the KRRCint key and the integrity protection algorithm received in the SrbSgn_ShortMacVerificationReq.  5G-L2-SRB shall set all input bits for COUNT, BEARER and DIRECTION to binary ones for the generation of MAC-I and shall use the 16 least significant bits of the output MAC-I for comparison for the integrity verification. 2. Prepares and sends a SrbSgnl_ShortMacVerificationResp. Note: De-ciphering is not applied during Short MAC-I verification while invoking 5G_L2_2800_replaced( 11240989 ). (ID: 11240965)
